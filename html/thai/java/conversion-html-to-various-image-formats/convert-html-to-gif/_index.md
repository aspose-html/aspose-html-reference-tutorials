## Converting HTML to GIF

ด้านล่างเป็นกระบวนการทำงานที่สมบูรณ์และสามารถรันได้ แต่ละขั้นตอนอธิบายด้วยภาษาง่าย ๆ เพื่อให้คุณปรับใช้กับโปรเจกต์ของตนเองได้

### ขั้นตอนที่ 1: โหลดเอกสาร HTML
สร้างอินสแตนซ์ `HTMLDocument` ที่ชี้ไปยังไฟล์ต้นทางของคุณ

```java
HTMLDocument htmlDocument = new HTMLDocument("your_input.html");
```

> **เคล็ดลับ:** หาก HTML ของคุณอ้างอิงทรัพยากรภายนอก (CSS, รูปภาพ) ให้วางไว้ในโฟลเดอร์เดียวกันหรือใช้ URL แบบเต็มเพื่อให้เรนเดอร์สามารถแก้ไขได้อย่างถูกต้อง

### ขั้นตอนที่ 2: ตั้งค่ารูปแบบผลลัพธ์
กำหนด `ImageSaveOptions` เพื่อบอก Aspose.HTML ว่าต้องการรูปแบบ GIF

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Gif);
```

คุณยังสามารถปรับคุณสมบัติเช่นคุณภาพของภาพ, สีพื้นหลัง, หรือระยะเวลาหน่วงของเฟรมได้ที่นี่หากต้องการ GIF แอนิเมชัน

### ขั้นตอนที่ 3: ระบุเส้นทางไฟล์ผลลัพธ์
กำหนดตำแหน่งที่ต้องการบันทึกไฟล์ GIF ที่ได้

```java
String outputFile = "output.gif";
```

### ขั้นตอนที่ 4: ดำเนินการแปลง
เมธอด `Converter.convertHTML` จะทำงานหนักทั้งหมดให้คุณ

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

หลังจากบรรทัดนี้ทำงานเสร็จ `output.gif` จะมีสแนปช็อตแบบเรนเดอร์ของหน้า HTML ต้นฉบับ

## ปัญหาที่พบบ่อยและวิธีแก้

- **CSS หรือรูปภาพหาย** – ตรวจสอบให้แน่ใจว่าเส้นทางสัมพันธ์ทั้งหมดถูกต้องหรือใช้ URL แบบเต็ม  
- **หน้า HTML ขนาดใหญ่** – เพิ่มการจัดสรรหน่วยความจำให้กับ JVM (`-Xmx`) หากพบ `OutOfMemoryError`  
- **คุณสมบัติ CSS ที่ไม่รองรับ** – Aspose.HTML ปฏิบัติตามมาตรฐานสมัยใหม่ส่วนใหญ่ แต่คุณสมบัติ CSS3 ที่ใหม่มากอาจถูกละเลย; พิจารณาให้งานสไตล์ชีตง่ายขึ้นสำหรับการแปลง
