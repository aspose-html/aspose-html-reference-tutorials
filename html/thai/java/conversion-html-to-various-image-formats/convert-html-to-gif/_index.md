## การแปลง HTML เป็น GIF
ด้านล่างเป็นกระบวนการที่สมบูรณ์และสามารถรันได้ แต่ละขั้นตอนอธิบายด้วยภาษาง่าย ๆ เพื่อให้คุณปรับใช้กับโครงการของตนเอง

### ขั้นตอนที่ 1: โหลดเอกสาร HTML
`HTMLDocument` เป็นอ็อบเจ็กต์ระดับบนของ Aspose.HTML ที่แสดงถึงไฟล์ HTML เดียวในหน่วยความจำ มันทำการพาร์สมาร์กอัป, แก้ไขทรัพยากร, และเตรียมเครื่องยนต์การจัดวาง

สร้างอินสแตนซ์ `HTMLDocument` ที่ชี้ไปยังไฟล์ต้นทางของคุณ

```java
HTMLDocument htmlDocument = new HTMLDocument("your_input.html");
```

> **เคล็ดลับ:** หาก HTML ของคุณอ้างอิงทรัพยากรภายนอก (CSS, images), วางไว้ในโฟลเดอร์เดียวกันหรือให้ URL แบบเต็มเพื่อให้เรนเดอร์สามารถแก้ไขได้อย่างถูกต้อง.

### ขั้นตอนที่ 2: ตั้งค่ารูปแบบผลลัพธ์
`ImageSaveOptions` กำหนดวิธีการบันทึกหน้าที่เรนเดอร์เป็นภาพ โดยการตั้งค่า property `format` เป็น `ImageFormat.Gif` คุณบอก Aspose.HTML ให้สร้างไฟล์ GIF

กำหนดค่า `ImageSaveOptions` เพื่อบอก Aspose.HTML ว่ารูปแบบเป้าหมายคือ GIF

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Gif);
```

คุณยังสามารถปรับแต่ง property เช่น คุณภาพภาพ, สีพื้นหลัง, หรือความหน่วงของเฟรมได้ที่นี่ หากต้องการ GIF แบบเคลื่อนไหว.

### ขั้นตอนที่ 3: กำหนดเส้นทางไฟล์ผลลัพธ์
ระบุที่ที่ต้องการบันทึก GIF ที่ได้ ผลลัพธ์เส้นทางอาจเป็นแบบเต็มหรือแบบสัมพันธ์กับไดเรกทอรีทำงานของโครงการของคุณ

กำหนดเส้นทางไฟล์ผลลัพธ์สำหรับภาพ GIF

```java
String outputFile = "output.gif";
```

### ขั้นตอนที่ 4: ดำเนินการแปลง
`Converter.convertHTML` เป็นเมธอดเดียวที่เรนเดอร์เอกสาร HTML และเขียนไฟล์ภาพตามตัวเลือกที่ให้ มันจัดการการโหลดทรัพยากร, การคำนวณการจัดวาง, และการแรสเตอร์ไลเซชันภายใน

เมธอด `Converter.convertHTML` ทำงานหนักทั้งหมด

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

หลังจากบรรทัดนี้ทำงาน `output.gif` จะมีสแนปช็อตที่แรสเตอร์ไลซ์ของหน้า HTML ดั้งเดิม
