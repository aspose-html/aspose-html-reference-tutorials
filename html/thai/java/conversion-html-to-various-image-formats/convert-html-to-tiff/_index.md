# HTML to Image Java – แปลง HTML เป็น TIFF ด้วย Aspose.HTML

ถ้าคุณต้องการ **html to image java** คุณมาถูกที่แล้ว ในบทแนะนำนี้เราจะอธิบายขั้นตอนการแปลงไฟล์ HTML ให้เป็นภาพ TIFF คุณภาพสูงโดยใช้ Aspose.HTML for Java วิธีเดียวกันนี้ยังทำงานได้กับ PNG, JPEG, BMP และรูปแบบเรสเตอร์อื่น ๆ ให้คุณมีโซลูชันที่ยืดหยุ่นซึ่งสามารถนำกลับมาใช้ใหม่ได้ในเครื่องมือรายงาน ระบบจัดเก็บเอกสาร หรือเครื่องสร้างภาพย่อ คุณจะได้เห็นว่าทำไมไลบรารีนี้จึงให้ผลลัพธ์ที่พิกเซล‑เพอร์เฟกต์และวิธีการผสานรวมเข้ากับโปรเจกต์ Java ใด ๆ

- **Which Java versions are supported?** Java 8 และรุ่นต่อไป.  
- **Is the code thread‑safe?** ใช่, API สามารถใช้ในสภาพแวดล้อมหลายเธรดได้.

Aspose.HTML จะทำการพาร์สมาร์กอัป, ประมวลผล CSS, รัน JavaScript, แล้วแปลงเลย์เอาต์สุดท้ายเป็นบิตแมพที่สามารถบันทึกเป็น TIFF, PNG, JPEG, BMP หรือรูปแบบอื่น ๆ ได้ ซึ่งทำให้คุณสามารถสร้างภาพย่อ, ใบแจ้งหนี้ที่พิมพ์ได้, หรือสแนปช็อตเพื่อเก็บรักษาโดยไม่ต้องใช้เบราว์เซอร์

1. **Java Development Environment**  
   ติดตั้ง Java Development Kit (JDK). คุณสามารถดาวน์โหลดได้จาก [Oracle website](https://www.oracle.com/java/technologies/javase-downloads.html).

2. **Aspose.HTML for Java**

3. **HTML Document**  
   เตรียมไฟล์ HTML ที่ต้องการแปลงไว้บนดิสก์ ซึ่งไฟล์นี้จะเป็นแหล่งข้อมูลสำหรับการแปลงเป็นภาพ

## นำเข้าแพ็กเกจ

ในโปรเจกต์ Java ของคุณ ให้นำเข้าคลาสสำคัญของ Aspose.HTML:

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
```

การนำเข้าดังกล่าวทำให้คุณเข้าถึงการโหลดเอกสาร, ตัวเลือกการบันทึกภาพ, และเอนจินการแปลงได้

## วิธีโหลดเอกสาร HTML ด้วย Java

โหลดไฟล์ต้นฉบับด้วยบรรทัดโค้ดเดียว คลาส `HTMLDocument` จะพาร์สมาร์กอัป, ประมวลผล CSS, และเตรียมหน้าเว็บสำหรับการเรนเดอร์ โดยอัตโนมัติจะตรวจจับการเข้ารหัสอักขระ, ติดตาม URL แบบสัมพันธ์, และโหลดทรัพยากรภายนอกเช่นรูปภาพและฟอนต์ เพื่อให้ผลลัพธ์ที่เรนเดอร์ตรงกับการแสดงผลในเบราว์เซอร์ ทำให้กระบวนการแปลงเป็นเรื่องง่ายและเชื่อถือได้

```java
HTMLDocument htmlDocument = new HTMLDocument("path/to/your/input.html");
```

เปลี่ยน `"path/to/your/input.html"` ให้เป็นเส้นทางจริงของไฟล์ HTML ของคุณ API จะตรวจจับการเข้ารหัสอักขระโดยอัตโนมัติและจัดการทรัพยากรภายนอกเช่นรูปภาพหรือฟอนต์ที่อ้างอิงใน HTML

## วิธีแปลง HTML เป็น TIFF (render html tiff)

กำหนดค่าการแปลงเพื่อสร้างภาพ TIFF enum `ImageFormat` ระบุประเภทภาพเอาต์พุต เช่น TIFF, PNG หรือ JPEG และให้คุณเลือกการบีบอัดที่เหมาะสม TIFF มักถูกเลือกเนื่องจากการบีบอัดแบบไม่มีการสูญเสียและรองรับเอกสารหลายหน้า ทำให้เหมาะสำหรับการเก็บรักษาและการพิมพ์ ปรับค่า DPI และความลึกสีเพื่อเพิ่มคุณภาพได้อีก

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Tiff);
```

หากคุณต้องการ **html to png java** ในภายหลัง เพียงเปลี่ยน `ImageFormat.Tiff` เป็น `ImageFormat.Png` เอนจินการแปลงจะรักษาคุณภาพเวกเตอร์เดิมไว้ ทำให้ภาพที่ได้ตรงกับการเรนเดอร์บนหน้าจอ

## วิธีตั้งค่าเส้นทางไฟล์ผลลัพธ์

กำหนดตำแหน่งที่ภาพที่สร้างจะถูกบันทึก คลาส `File` แทนที่เส้นทางไฟล์หรือไดเรกทอรีในระบบไฟล์ ทำให้คุณระบุตำแหน่งที่แน่นอนสำหรับเอาต์พุต `File` จะชี้ไปยังตำแหน่งบนไฟล์ซิสเต็มของคุณ และไลบรารีจะสร้างไดเรกทอรีที่ขาดหายโดยอัตโนมัติ คุณยังสามารถตั้งค่าการเขียนได้หากต้องการ

```java
String outputFile = "path/to/your/output.tif";
```

ปรับนามสกุลไฟล์หากเลือกรูปแบบอื่น; วิธี `save` เดียวกันทำงานได้กับ PNG, JPEG, BMP, และ GIF

## วิธีทำการแปลง

เรียกการแปลงด้วย API เพียงครั้งเดียว เมธอด `save` จะเขียนภาพที่แรสเตอร์ลงในเส้นทางเป้าหมายและปล่อยทรัพยากรเนทีฟทั้งหมด คลาส `ImageSaveOptions` ให้คุณระบุพารามิเตอร์เอาต์พุตเช่นรูปแบบ, ความละเอียด, และการบีบอัด มันรับอ็อบเจ็กต์ `File` ปลายทางและ `ImageSaveOptions` แล้วใช้รูปแบบและการตั้งค่าคุณภาพที่เลือก หลังจากเมธอดทำงานเสร็จ เอกสารจะถูกปิดและหน่วยความจำจะถูกคืนค่า เพื่อให้การประมวลผลแบบแบตช์มีประสิทธิภาพ

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

หลังจากดำเนินการ คุณจะพบไฟล์ TIFF ที่ตำแหน่งที่ระบุ การทำงานเสร็จในไม่กี่มิลลิวินาทีสำหรับ HTML ขนาดหน้าเว็บทั่วไป และสเกลเชิงเส้นสำหรับเอกสารที่ใหญ่ขึ้น
