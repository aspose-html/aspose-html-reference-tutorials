# แปลง HTML เป็น BMP ด้วย Aspose.HTML สำหรับ Java

หากคุณต้องการ **convert html to bmp** อย่างรวดเร็วและเชื่อถือได้ คุณอยู่ในสถานที่ที่ถูกต้อง ในบทแนะนำนี้เราจะพาคุณผ่านทุกอย่างที่คุณต้องการ—ตั้งแต่การตั้งค่าสภาพแวดล้อมการพัฒนาไปจนถึงการเขียนโค้ด Java ที่แปลงไฟล์ HTML ให้เป็นภาพ BMP คุณภาพสูง เมื่อเสร็จสิ้นคุณจะเข้าใจไม่เพียง *วิธีแปลง html* แต่ยังรู้ว่าทำไมวิธีนี้จึงเหมาะกับสถานการณ์การเรนเดอร์ฝั่งเซิร์ฟเวอร์ด้วย Java

## คำตอบด่วน
- **What does the conversion produce?** ภาพ BMP แบบแรสเตอร์ที่คงรูปแบบการแสดงผลของ HTML ต้นฉบับไว้  
- **Which library is required?** Aspose.HTML for Java (รองรับ BMP, PNG, JPEG ฯลฯ).  
- **Do I need a license?** ใบอนุญาตประเมินชั่วคราวใช้ได้สำหรับการทดสอบ; จำเป็นต้องมีใบอนุญาตเต็มสำหรับการใช้งานจริง.  
- **Can I run this on any OS?** ได้—Java เป็นแบบข้ามแพลตฟอร์ม ดังนั้นโค้ดสามารถทำงานบน Windows, Linux หรือ macOS.  
- **How long does the conversion take?** โดยทั่วไปใช้เวลาน้อยกว่าวินาทีสำหรับหน้าแบบมาตรฐาน; หน้าใหญ่กว่าอาจใช้เวลาสองสามวินาที.  

## convert html to bmp คืออะไร?
`convert html to bmp` คือกระบวนการเรนเดอร์เอกสาร HTML ให้เป็นภาพ bitmap (BMP) การแปลงจะจับภาพการจัดวางที่แม่นยำ รวมถึงข้อความ สี ฟอนต์ และกราฟิกเวกเตอร์ และบันทึกเป็นไฟล์ภาพแรสเตอร์ ซึ่งทำให้เนื้อหา HTML สามารถแสดงในสภาพแวดล้อมที่รองรับเฉพาะรูปแบบ bitmap เท่านั้น โดยคงลักษณะเดิมของมัน

## วิธีแปลง HTML เป็น BMP ด้วย Aspose.HTML?
เพื่อแปลง HTML เป็น BMP ด้วย Aspose.HTML ก่อนอื่นให้โหลดไฟล์ HTML เข้าไปใน `HTMLDocument` จากนั้นสร้างอ็อบเจ็กต์ `ImageSaveOptions` ที่ระบุรูปแบบ BMP และการตั้งค่าความละเอียดตามต้องการ จากนั้นกำหนดเส้นทางไฟล์ผลลัพธ์ที่บิตแมพจะถูกเขียน สุดท้ายเรียก `Document.save` พร้อมตัวเลือกเพื่อเรนเดอร์และบันทึกภาพ เวิร์กโฟลว์นี้จัดการ CSS, ฟอนต์ และ SVG โดยอัตโนมัติ

ด้านล่างเป็นคู่มือสั้น ๆ แบบลำดับเลขที่พาคุณผ่านแต่ละขั้นตอน โค้ดบล็อกเหมือนกับในบทแนะนำต้นฉบับ; เราเพียงเพิ่มบริบทและคำอธิบาย

### ขั้นตอนที่ 1: นำเข้าแพ็กเกจ Aspose.HTML สำหรับ Java

`HTMLDocument` คือคลาสหลักที่แสดงไฟล์ HTML ที่โหลดเข้าสู่หน่วยความจำเพื่อการเรนเดอร์.  
```java
// Source HTML document
com.aspose.html.HTMLDocument htmlDocument = new com.aspose.html.HTMLDocument("path/to/your/input.html");
```

เราสร้างอินสแตนซ์ `HTMLDocument` ที่แสดง HTML ที่คุณต้องการเรนเดอร์ แทนที่ `"path/to/your/input.html"` ด้วยตำแหน่งไฟล์จริง

### ขั้นตอนที่ 2: เริ่มต้น ImageSaveOptions สำหรับ BMP

`ImageSaveOptions` กำหนดรูปแบบแรสเตอร์, ความละเอียด, และการตั้งค่าคุณภาพสำหรับภาพผลลัพธ์.  
```java
// Initialize ImageSaveOptions
com.aspose.html.saving.ImageSaveOptions options = new com.aspose.html.saving.ImageSaveOptions(com.aspose.html.rendering.image.ImageFormat.Bmp);
```

`ImageSaveOptions` บอก Aspose.HTML ว่าจะสร้างรูปแบบแรสเตอร์ใด ที่นี่เรากำหนดเป็น `Bmp` แต่คุณสามารถเปลี่ยนเป็น PNG, JPEG ฯลฯ หากต้องการรูปแบบ **java html to image** ที่แตกต่างในภายหลัง

### ขั้นตอนที่ 3: กำหนดเส้นทางไฟล์ผลลัพธ์

`OutputPath` เป็นสตริงง่าย ๆ ที่บอกไลบรารีว่าจะเขียนไฟล์ BMP ที่สร้างไว้ที่ไหน.  
```java
// Output file path
String outputFile = "path/to/your/output/HTMLtoBMP_Output.bmp";
```

ตั้งค่าที่เก็บไฟล์ BMP ที่จะบันทึก ปรับเส้นทางตามที่จำเป็นสำหรับโครงสร้างโปรเจกต์ของคุณ

### ขั้นตอนที่ 4: ทำการแปลง

เมธอด `Document.save` เขียนเนื้อหาที่เรนเดอร์ลงไฟล์โดยใช้ตัวเลือกการบันทึกที่ให้ไว้.  
`Document.save` ทำงานของ pipeline การเรนเดอร์, แปลงเอกสาร HTML ให้เป็นภาพ BMP ที่คุณระบุ.  
```java
// Convert HTML to BMP
com.aspose.html.converters.Converter.convertHTML(htmlDocument, options, outputFile);
```
