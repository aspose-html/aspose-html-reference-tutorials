**Q:** *ฉันสามารถแปลงไฟล์ EPUB ที่มีการป้องกันเป็น PDF ได้หรือไม่?*  
**A:** ได้. Aspose.HTML for Java สามารถเปิด EPUB ที่เข้ารหัสได้เมื่อคุณให้รหัสผ่านการถอดรหัสที่เหมาะสม  

**Q:** *ถ้าฉันต้องการให้ PDF เป็นไปตามมาตรฐาน PDF/A จะทำอย่างไร?*  
**A:** ใช้คลาส `PdfSaveOptions` เพื่อตั้งค่า `Compliance` เป็น `PdfCompliance.PdfA1b` (หรือระดับ PDF/A อื่น) ก่อนบันทึก  

**Q:** *สามารถดึงภาพเช่น JPG หรือ PNG ระหว่างการแปลงได้หรือไม่?*  
**A:** แน่นอน. โดยการระบุ image save option หรือใช้ custom stream provider คุณสามารถกำหนดให้แต่ละหน้า หรือภาพ ไปเป็นรูปแบบ JPG หรือ PNG  

**Q:** *ฉันต้องการไลบรารีแยกต่างหากสำหรับการดึงภาพหรือไม่?*
