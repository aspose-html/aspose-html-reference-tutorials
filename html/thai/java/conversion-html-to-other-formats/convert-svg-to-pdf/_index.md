## วิธีสร้าง PDF จาก SVG ด้วย Aspose.HTML for Java

### Import Packages
ก่อนอื่น ให้นำเข้าคลาสที่จำเป็นเข้าสู่โปรเจกต์ Java ของคุณ

```java
import com.aspose.html.dom.svg.SVGDocument;
import com.aspose.html.saving.PdfSaveOptions;
import com.aspose.html.converters.Converter;
```

### ขั้นตอนที่ 1: โหลดเอกสาร SVG
เพื่อเริ่มการแปลง โหลดไฟล์ SVG ต้นฉบับเข้าสู่วัตถุ `SVGDocument` นี่เป็นการสาธิต **วิธีโหลดเอกสาร SVG แบบ Java**

```java
SVGDocument svgDocument = new SVGDocument(Resources.input("input.svg"));
```

### ขั้นตอนที่ 2: ตั้งค่าตัวเลือกการบันทึก PDF
กำหนดตัวเลือกการส่งออก PDF ตัวอย่างเช่น คุณสามารถกำหนดคุณภาพ JPEG เพื่อให้ได้ผลลัพธ์ภาพที่ดีที่สุดเมื่อทำการเรสเตอร์ไอเมจภายใน SVG

```java
PdfSaveOptions options = new PdfSaveOptions();
options.setJpegQuality(100);
```

### ขั้นตอนที่ 3: กำหนดเส้นทางไฟล์ผลลัพธ์
ระบุที่ที่คุณต้องการบันทึก PDF ที่สร้างขึ้น ตรวจสอบให้แน่ใจว่าไดเรกทอรีมีอยู่และคุณมีสิทธิ์เขียน

```java
String outputFile = Resources.output("SVGtoPDF_Output.pdf");
```

### ขั้นตอนที่ 4: แปลง SVG เป็น PDF
สุดท้าย ให้ทำการแปลงโดยเรียกเมธอด `convertSVG` ขั้นตอนนี้ **แปลงกราฟิกเวกเตอร์ให้พร้อมเป็น PDF**

```java
Converter.convertSVG(svgDocument, options, outputFile);
```

ยินดีด้วย! คุณได้ **สร้าง PDF จาก SVG** ด้วย Aspose.HTML for Java สำเร็จแล้ว PDF ที่ได้จะอยู่ที่เส้นทางที่คุณกำหนดใน `outputFile`
