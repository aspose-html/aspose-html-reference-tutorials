โหลด SVG, ตั้งค่าตัวเลือก PDF, และบันทึกผลลัพธ์ – ทั้งหมดในสี่ขั้นตอนสั้น ๆ.

### คำตอบโดยตรง
โหลด SVG ของคุณด้วย `new SVGDocument("input.svg")`, ตั้งค่า `PdfSaveOptions` (เช่น ตั้ง `jpegQuality` เป็น 90), จากนั้นเรียก `document.save("output.pdf", saveOptions)`. กระบวนการบรรทัดเดียวนี้จะแปลงกราฟิกเวกเตอร์เป็น PDF พร้อมคงการจัดวาง, สี, และฟอนต์.

### นำเข้าแพ็กเกจ
คลาส `SVGDocument` เป็นการแสดงผลของไฟล์ SVG ในหน่วยความจำของ Aspose.HTML. นำเข้าชื่อเนมสเปซที่จำเป็นก่อนเริ่ม:

**Definition anchor:** `SVGDocument` คือคลาสหลักที่พาร์สและเก็บเนื้อหา SVG เพื่อการประมวลผลต่อไป.

```
import com.aspose.html.SVGDocument;
import com.aspose.html.rendering.pdf.PdfSaveOptions;
import com.aspose.html.rendering.pdf.PdfDevice;
```

### ขั้นตอนที่ 1: โหลดเอกสาร SVG
`SVGDocument` อ่านไฟล์ SVG ที่เป็น XML และเตรียมพร้อมสำหรับการเรนเดอร์.

**Definition anchor:** `SVGDocument` โหลดแหล่ง SVG และให้ API แบบ DOM สำหรับการจัดการ.

```
SVGDocument svgDoc = new SVGDocument("path/to/input.svg");
```

### ขั้นตอนที่ 2: ตั้งค่า PDF Save Options
`PdfSaveOptions` ให้คุณควบคุมคุณภาพผลลัพธ์, ขนาดหน้า, และการบีบอัด.

**Definition anchor:** `PdfSaveOptions` รวมการตั้งค่าเฉพาะ PDF เช่น คุณภาพภาพและขนาดหน้า.

```
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.setJpegQuality(90);               // Set JPEG quality to 90%
saveOptions.setPageSize(PdfDevice.PageSize.A4); // Use A4 page size
```

### ขั้นตอนที่ 3: กำหนดเส้นทางเอาต์พุต
เลือกโฟลเดอร์ที่สามารถเขียนได้และชื่อไฟล์สำหรับ PDF ที่สร้าง.

```
String outputPath = "path/to/output.pdf";
```

### ขั้นตอนที่ 4: แปลง SVG เป็น PDF
เมธอด `save` ทำการแปลงจริง.

**Definition anchor:** `document.save` เขียนเนื้อหาที่เรนเดอร์ไปยังรูปแบบที่ระบุโดยใช้ตัวเลือกที่ให้มา.

```
svgDoc.save(outputPath, saveOptions);
```
