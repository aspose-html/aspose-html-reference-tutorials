## Cómo **convert html gif**, **convert html jpeg** y **convert html png** con Aspose.HTML
A continuación encontrará instrucciones concisas paso a paso para cada formato de destino. Los fragmentos de código son idénticos excepto por la extensión del archivo de salida; solo necesita cambiar el formato de `ImageSaveOptions`.

### Paso 1: Configurar el renderizador HTML
Cree una instancia de `HtmlRenderer` y cargue la fuente HTML (archivo, URL o cadena).

### Paso 2: Elegir el formato de imagen deseado
Seleccione una de las subclases de `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions`, etc.

### Paso 3: Renderizar y guardar
Llame a `renderer.renderToFile(outputPath, saveOptions)` para generar la imagen.

*Los mismos tres pasos se aplican a BMP, TIFF y cualquier otro formato compatible.*

## Consejos de solución de problemas
- **Salida en blanco** – Asegúrese de que la fuente HTML esté completamente cargada; use `renderer.waitForComplete()` si es necesario.

- **Fuentes faltantes** – Instale las fuentes requeridas en el servidor o incrústelas en el HTML usando `@font-face`.
