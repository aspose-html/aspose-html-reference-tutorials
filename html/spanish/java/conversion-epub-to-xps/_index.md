## Errores comunes y consejos
- **Imágenes grandes** – Si su EPUB contiene imágenes de alta resolución, considere reducir la muestra antes de la conversión para mantener un tamaño razonable del archivo XPS.  
- **Incrustación de fuentes** – Asegúrese de que las fuentes requeridas estén disponibles en el sistema; de lo contrario, Aspose.HTML podría sustituirlas, afectando el diseño.  
- **Uso de memoria** – Para EPUBs muy grandes, procese las páginas de forma incremental para evitar un consumo excesivo de memoria.
