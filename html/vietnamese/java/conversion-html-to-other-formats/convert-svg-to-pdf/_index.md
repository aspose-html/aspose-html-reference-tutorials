Để chuyển đổi một tệp SVG sang PDF với Aspose.HTML for Java, bạn tải SVG vào một `SVGDocument`, cấu hình `PdfSaveOptions` để kiểm soát chất lượng và bố cục, sau đó gọi phương thức `save` để ghi PDF. Tải SVG, cấu hình tùy chọn PDF, và lưu kết quả — tất cả trong bốn bước ngắn gọn.

### Câu trả lời trực tiếp
Tải SVG của bạn bằng `new SVGDocument("input.svg")`, cấu hình `PdfSaveOptions` (ví dụ, đặt `jpegQuality` thành 90), sau đó gọi `document.save("output.pdf", saveOptions)`. Quy trình một dòng này chuyển đổi đồ họa vector sang PDF trong khi vẫn giữ nguyên bố cục, màu sắc và phông chữ.

### Nhập gói
Lớp `SVGDocument` là đại diện của Aspose.HTML cho một tệp SVG trong bộ nhớ. Nhập các namespace cần thiết trước khi bắt đầu:

**Definition anchor:** `SVGDocument` là lớp cốt lõi phân tích và giữ nội dung SVG để xử lý tiếp.

```
import com.aspose.html.SVGDocument;
import com.aspose.html.rendering.pdf.PdfSaveOptions;
import com.aspose.html.rendering.pdf.PdfDevice;
```

### Bước 1: Tải tài liệu SVG
`SVGDocument` đọc tệp SVG dựa trên XML và chuẩn bị nó để render.

**Definition anchor:** `SVGDocument` loads the SVG source and provides a DOM‑like API for manipulation.

```
SVGDocument svgDoc = new SVGDocument("path/to/input.svg");
```

### Bước 2: Cấu hình tùy chọn lưu PDF
`PdfSaveOptions` cho phép bạn kiểm soát chất lượng đầu ra, kích thước trang và nén.

**Definition anchor:** `PdfSaveOptions` encapsulates all PDF‑specific settings such as image quality and page dimensions.

```
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.setJpegQuality(90);               // Set JPEG quality to 90%
saveOptions.setPageSize(PdfDevice.PageSize.A4); // Use A4 page size
```

### Bước 3: Xác định đường dẫn đầu ra
Chọn một thư mục có thể ghi và tên tệp cho PDF được tạo.

```
String outputPath = "path/to/output.pdf";
```

### Bước 4: Chuyển SVG sang PDF
Phương thức `save` thực hiện việc chuyển đổi thực tế.

**Definition anchor:** `document.save` writes the rendered content to the specified format using the supplied options.

```
svgDoc.save(outputPath, saveOptions);
```

Chúc mừng! Bạn đã **tạo PDF từ SVG** thành công bằng Aspose.HTML for Java. PDF sẽ được lưu tại đường dẫn bạn cung cấp trong `outputPath`.
