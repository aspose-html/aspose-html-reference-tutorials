## Các vấn đề thường gặp và giải pháp
- **Missing fonts in PDF** – Đảm bảo các phông chữ cần thiết đã được cài đặt trên máy chủ hoặc nhúng chúng bằng `PdfSaveOptions`.  
- **Large EPUB files cause memory pressure** – Sử dụng xử lý dựa trên stream (`MemoryStream` → `FileStream`) để giảm sử dụng heap.  
- **Canvas rendering appears blank** – Kiểm tra rằng canvas đã được vẽ hoàn toàn trước khi gọi API chuyển đổi.
