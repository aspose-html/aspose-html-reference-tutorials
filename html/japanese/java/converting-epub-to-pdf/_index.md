**Q:** *保護された EPUB ファイルを PDF に変換できますか？*  
**A:** はい。Aspose.HTML for Java は、適切な復号パスワードを提供すれば暗号化された EPUB を開くことができます。

**Q:** *PDF を PDF/A に準拠させる必要がある場合は？*  
**A:** `PdfSaveOptions` クラスを使用して、`Compliance` を `PdfCompliance.PdfA1b`（または他の PDF/A レベル）に設定してから保存します。

**Q:** *変換中に JPG や PNG などの画像を抽出することは可能ですか？*

**Q:** *画像抽出のために別のライブラリが必要ですか？*  
**A:** いいえ。同じ Aspose.HTML for Java API が PDF 生成と画像抽出の両方を処理し、ワークフローを簡素化します。
