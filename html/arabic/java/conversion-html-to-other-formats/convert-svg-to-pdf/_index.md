## كيفية إجراء تحويل svg إلى pdf java باستخدام Aspose.HTML للـ Java
لتحويل ملف SVG إلى PDF باستخدام Aspose.HTML للـ Java، تقوم بتحميل الـ SVG في `SVGDocument`، وتكوين `PdfSaveOptions` للتحكم في الجودة والتخطيط، ثم تستدعي طريقة `save` لكتابة ملف PDF. يمكن تغليف سير العمل هذا داخل حلقات للمعالجة الدفعية ودمجه في أي تطبيق Java.

قم بتحميل الـ SVG، وتكوين خيارات PDF، وحفظ النتيجة — كل ذلك في أربع خطوات مختصرة.

### إجابة مباشرة
حمّل الـ SVG الخاص بك باستخدام `new SVGDocument("input.svg")`، قم بتكوين `PdfSaveOptions` (مثلاً، اضبط `jpegQuality` إلى 90)، ثم استدعِ `document.save("output.pdf", saveOptions)`. هذه العملية ذات السطر الواحد تحول الرسوم المتجهة إلى PDF مع الحفاظ على التخطيط والألوان والخطوط.

### استيراد الحزم
فئة `SVGDocument` هي تمثيل Aspose.HTML لملف SVG في الذاكرة. استورد المساحات الاسمية اللازمة قبل البدء:

**Definition anchor:** `SVGDocument` هي الفئة الأساسية التي تقوم بتحليل وحفظ محتوى SVG للمعالجة اللاحقة.

```
import com.aspose.html.SVGDocument;
import com.aspose.html.rendering.pdf.PdfSaveOptions;
import com.aspose.html.rendering.pdf.PdfDevice;
```

### الخطوة 1: تحميل مستند SVG
`SVGDocument` يقرأ ملف SVG القائم على XML ويجهزه للعرض.

**Definition anchor:** `SVGDocument` يقوم بتحميل مصدر SVG ويوفر واجهة شبيهة بـ DOM للتلاعب.

```
SVGDocument svgDoc = new SVGDocument("path/to/input.svg");
```

### الخطوة 2: تكوين خيارات حفظ PDF
`PdfSaveOptions` يتيح لك التحكم في جودة الإخراج، حجم الصفحة، والضغط.

**Definition anchor:** `PdfSaveOptions` يجمع جميع إعدادات PDF الخاصة مثل جودة الصورة وأبعاد الصفحة.

```
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.setJpegQuality(90);               // Set JPEG quality to 90%
saveOptions.setPageSize(PdfDevice.PageSize.A4); // Use A4 page size
```

### الخطوة 3: تحديد مسار الإخراج
اختر مجلدًا قابلًا للكتابة واسم ملف للـ PDF المُنشأ.

```
String outputPath = "path/to/output.pdf";
```

### الخطوة 4: تحويل SVG إلى PDF
طريقة `save` تقوم بالتحويل الفعلي.

**Definition anchor:** `document.save` يكتب المحتوى المُعرض إلى الصيغة المحددة باستخدام الخيارات المقدمة.

```
svgDoc.save(outputPath, saveOptions);
```

تهانينا! لقد نجحت في **generated a PDF from SVG** باستخدام Aspose.HTML للـ Java. سيقع ملف PDF في المسار الذي قدمته في `outputPath`.
