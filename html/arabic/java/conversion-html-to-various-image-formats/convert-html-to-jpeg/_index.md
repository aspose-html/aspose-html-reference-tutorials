# كيف تحول HTML إلى JPEG باستخدام Aspose.HTML للـ Java

## المقدمة

إذا كنت بحاجة إلى تحويل صفحة ويب أو أي علامة HTML إلى صورة JPEG ثابتة، فأنت في المكان الصحيح. في هذا الدرس سنوضح **كيفية تحويل HTML إلى JPEG** باستخدام Aspose.HTML للـ Java، مع تغطية كل شيء من إعداد بيئتك إلى ضبط صورة الإخراج بدقة. بنهاية الدليل ستتمكن من دمج تحويل HTML إلى صورة في تطبيقات Java الخاصة بك ببضع أسطر من الشيفرة فقط.

## إجابات سريعة
- **ما المكتبة التي تقوم بالتحويل؟** Aspose.HTML للـ Java

- **هل أحتاج إلى ترخيص؟** يلزم وجود ترخيص تجاري للإنتاج؛ يتوفر نسخة تجريبية مجانية

## استيراد الحزم

أول شيء نحتاج إلى القيام به هو جلب الفئات المطلوبة من Aspose.HTML إلى مشروعنا. هذه الخطوة تضمن أن المترجم يعرف أين يجد واجهات برمجة التحويل.

### الخطوة 1: استيراد حزم Aspose.HTML

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
```

مع استيراد الحزم، نحن جاهزون لتحميل HTML وبدء التحويل.

## دليل التحويل خطوة بخطوة

فيما يلي نقسم العملية بالكامل إلى خطوات واضحة مرقمة. كل خطوة تتضمن شرحًا قصيرًا يليه الشيفرة الدقيقة التي تحتاج إلى نسخها.

### الخطوة 2: تحميل مستند HTML المصدر

أنشئ كائن `HTMLDocument` يشير إلى ملف الإدخال الخاص بك. تقوم Aspose.HTML بقراءة الملف، وتحليل العلامات، وبناء DOM جاهز للعرض.

```java
HTMLDocument htmlDocument = new HTMLDocument("input.html");
```

> **نصيحة:** استبدل `"input.html"` بالمسار المطلق أو النسبي لملف HTML الفعلي الخاص بك.

### الخطوة 3: إعداد ImageSaveOptions للـ JPEG

`ImageSaveOptions` تخبر المحول بصيغة الصورة التي يجب إنتاجها وتتيح لك تعديل معلمات مثل الجودة والدقة. هنا نحدد JPEG كصيغة الهدف.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Jpeg);
```

> **نصيحة احترافية:** إذا احتجت يومًا ما إلى إنشاء PNG بدلاً من ذلك، ما عليك سوى تغيير `ImageFormat.Jpeg` إلى `ImageFormat.Png`. هذا أيضًا يطابق الكلمة المفتاحية الثانوية **java convert html png**.

### الخطوة 4: تحديد مسار ملف الإخراج

حدد أين يجب حفظ صورة JPEG الناتجة. يمكنك تضمين أي مجلد تريده؛ فقط تأكد من أن التطبيق يمتلك صلاحيات الكتابة.

```java
String outputFile = "HTMLtoJPEG_Output.jpeg";
```

لا تتردد في إعادة تسمية الملف أو تغيير الامتداد إذا كنت تستهدف نوع صورة مختلف.

### الخطوة 5: تنفيذ التحويل

أخيرًا، استدعِ الطريقة الساكنة `convertHTML`. تأخذ المستند المحمَّل، خيارات الحفظ، ومسار الوجهة، ثم تكتب JPEG إلى القرص.

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

هذا كل شيء! عندما ينتهي البرنامج، ستجد صورة JPEG تمثل صفحة HTML الخاصة بك في الموقع الذي حددته.
