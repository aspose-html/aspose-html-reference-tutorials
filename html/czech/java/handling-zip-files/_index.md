## Jak odstranit soubory ze zipu pomocí Aspose.HTML pro Java

`ZIP Archive Message Handler` je třída, která poskytuje manipulaci se ZIP archivy v paměti. Načtěte ZIP pomocí **ZIP Archive Message Handler**, najděte položku, kterou chcete smazat, zavolejte metodu `remove` a poté archiv uložte. Tím se soubor odstraní bez rozbalení celého ZIP, což zkrátí I/O čas a udrží vaši aplikaci responzivní.

Aspose.HTML poskytuje **ZIP Archive Message Handler**, který funguje jako osobní asistent pro vaše komprimované balíčky. S několika voláními metod můžete otevřít ZIP, najít položku, kterou chcete smazat, a odstranit ji — vše bez ručního rozbalení archivu nejprve. Tento přístup šetří I/O režii a udržuje vaši aplikaci responzivní.

## Jak přidat soubory do zip java pomocí Aspose.HTML

Otevřete archiv pomocí handleru, zavolejte `add` (nebo `replace`) pro vložení nové položky a uložte změny. Handler aktualizuje ZIP v paměti, takže nikdy nemusíte archiv vytvářet znovu od začátku.

Stejný handler také podporuje scénáře **add files to zip java**. Po otevření archivu můžete vkládat nové položky nebo nahrazovat existující, což je ideální pro dynamické generování obsahu nebo aktualizace za běhu.
