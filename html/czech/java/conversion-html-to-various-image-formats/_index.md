Fráze *convert html gif* odkazuje na proces vykreslení HTML stránky a export vizuálního výsledku jako GIF obrázku. To je užitečné pro vytváření lehkých náhledů, animovaných snímků obrazovky nebo vkládání webového obsahu do dokumentů, které přijímají pouze soubory obrázků.

## Jak **convert html gif**, **convert html jpeg** a **convert html png** s Aspose.HTML
Níže najdete stručné kroky‑za‑krokem instrukce pro každý cílový formát. Ukázky kódu jsou identické kromě přípony výstupního souboru; stačí změnit formát v `ImageSaveOptions`.

### Krok 1: Nastavte HTML renderer
Vytvořte instanci `HtmlRenderer` a načtěte zdroj HTML (soubor, URL nebo řetězec).

### Krok 2: Vyberte požadovaný formát obrázku
Vyberte jednu z podtříd `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions` atd.

### Krok 3: Vykreslete a uložte
Zavolejte `renderer.renderToFile(outputPath, saveOptions)`, aby se obrázek vygeneroval.

*Stejné tři kroky platí pro BMP, TIFF a jakýkoli jiný podporovaný formát.*
