# Spuštění JavaScriptu v Javě – Kompletní průvodce přidáváním prvků do těla

Už jste někdy potřebovali **execute JavaScript in Java**, ale nebyli jste si jisti, jak propojit oba světy? Nejste v tom sami. Mnoho vývojářů Java narazí na problém, když chtějí provádět **JavaScript DOM manipulation** bez spouštění plnohodnotného prohlížeče. Dobrá zpráva? S Aspose.HTML můžete spustit skript, **append element to body**, a uložit upravený markup — vše z čistého Java kódu.

V tomto tutoriálu vás provede vším, co potřebujete vědět: načtení HTML souboru, napsání malého skriptu, který **modifies HTML with JavaScript**, spuštění tohoto skriptu z Javy a nakonec uložení změn. Na konci budete mít samostatný, spustitelný příklad, který můžete vložit do libovolného projektu.

## Co budete potřebovat

- **Java 8** nebo novější (API funguje s Java 8+)
- **Aspose.HTML for Java** knihovna (verze 22.12 nebo novější)
- Jednoduchý HTML šablonový soubor (vytvoříme jej za běhu)
- IDE nebo textový editor – nic zvláštního, i obyčejný textový editor stačí

> Pro tip: Pokud používáte Maven, přidejte závislost Aspose.HTML do svého `pom.xml`. Jinak stačí vložit JAR soubory do classpathu.

## Krok 1 – Načtení HTML dokumentu (Příprava na spuštění JavaScriptu v Javě)

Prvním krokem je vytvořit instanci `HTMLDocument`, která ukazuje na zdrojový soubor. Představte si tento objekt jako most mezi Javou a DOM stromem, který skript později upraví.

```java
import com.aspose.html.HTMLDocument;

// Step 1: Load the HTML document you want to manipulate
HTMLDocument htmlDoc = new HTMLDocument("YOUR_DIRECTORY/template.html");
```

> **Proč je to důležité:** Třída `HTMLDocument` parsuje soubor do DOM reprezentace, se kterou může pracovat Aspose skriptovací engine. Bez načtení dokumentu nejprve není pro skript nic, co by mohl upravit.

## Krok 2 – Napsání JavaScriptu, který **Append Element to Body**

Nyní vytvoříme malý úryvek, který vytvoří element `<p>` a přidá jej do `<body>`. Jedná se o klasickou **JavaScript DOM manipulation**, ale spustíme ji z Javy.

```java
// Step 2: JavaScript that adds a new paragraph to the body
String jsCode = ""
    + "var p = document.createElement('p');"
    + "p.textContent = 'Generated by JavaScript';"
    + "document.body.appendChild(p);";
```

> **Proč to funguje:** `document.createElement` vytvoří nový uzel, `textContent` nastaví jeho vnitřní text a `appendChild` jej připojí k existujícímu elementu `<body>`. Skript je čistý client‑side JavaScript; Aspose.HTML jej spouští v headless prostředí.

## Krok 3 – Vytvoření Script Engine (Engine, který **Run JavaScript from Java**)

Aspose.HTML obsahuje `ScriptEngine`, který dokáže vyhodnotit JavaScript vůči načtenému DOM. Je to část, která skutečně **run JavaScript from Java**.

```java
import com.aspose.html.scripting.ScriptEngine;

// Step 3: Bind the engine to the document we just loaded
ScriptEngine scriptEngine = new ScriptEngine(htmlDoc);
```

> **Proč to potřebujete:** Engine izoluje vykonávání skriptu, aby neovlivnil hlavní Java proces. Také vrací `ScriptExecutionResult`, abyste mohli ověřit úspěch.

## Krok 4 – Spuštění skriptu a kontrola výsledku

Po nastavení všeho konečně spustíme skript. Metoda `execute` vrací objekt výsledku, který nám říká, zda skript proběhl bez chyb.

```java
import com.aspose.html.scripting.ScriptExecutionResult;

// Step 4: Run the JavaScript and verify it succeeded
ScriptExecutionResult execResult = scriptEngine.execute(jsCode);
if (execResult.isSuccess()) {
    System.out.println("Script executed successfully.");
} else {
    System.err.println("Script failed: " + execResult.getErrorMessage());
}
```

> **Hraniční případ:** Pokud skript vyhodí výjimku (např. syntaktickou chybu), `isSuccess()` bude `false` a `getErrorMessage()` poskytne podrobnosti. V produkčním kódu to vždy kontrolujte.

## Krok 5 – Uložení upraveného dokumentu (Dokončení **Modify HTML with JavaScript**)

Po tom, co skript upraví DOM, uložíme změny zpět na disk. Výsledný soubor nyní obsahuje nový odstavec.

```java
// Step 5: Write the updated DOM back to an HTML file
htmlDoc.save("YOUR_DIRECTORY/modified.html");
System.out.println("Document saved.");
```

> **Ověření výsledku:** Otevřete `modified.html` v libovolném prohlížeči. Měli byste vidět původní obsah plus nový řádek s textem „Generated by JavaScript“.

## Krok 6 – Kompletní funkční příklad (Všechny kroky dohromady)

Níže je kompletní program, který můžete zkopírovat a vložit do souboru `JsDomManipulation.java`. Obsahuje všechny importy, zpracování chyb a komentáře pro přehlednost.

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.scripting.ScriptEngine;
import com.aspose.html.scripting.ScriptExecutionResult;

public class JsDomManipulation {
    public static void main(String[] args) throws Exception {

        // 1️⃣ Load the HTML document to be manipulated
        HTMLDocument htmlDoc = new HTMLDocument("YOUR_DIRECTORY/template.html");

        // 2️⃣ Define JavaScript that adds a new paragraph to the <body>
        String jsCode = "var p = document.createElement('p');"
                      + "p.textContent = 'Generated by JavaScript';"
                      + "document.body.appendChild(p);";

        // 3️⃣ Create a script engine bound to the loaded document
        ScriptEngine scriptEngine = new ScriptEngine(htmlDoc);

        // 4️⃣ Execute the JavaScript and report success
        ScriptExecutionResult execResult = scriptEngine.execute(jsCode);
        if (execResult.isSuccess()) {
            System.out.println("Script executed successfully.");
        } else {
            System.err.println("Script failed: " + execResult.getErrorMessage());
            return; // Stop if the script didn't run
        }

        // 5️⃣ Save the modified document back to HTML
        htmlDoc.save("YOUR_DIRECTORY/modified.html");
        System.out.println("Document saved.");
    }
}
```

### Očekávaný výstup

Running the program prints:

```
Script executed successfully.
Document saved.
```

And `modified.html` now contains something like:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Template</title>
</head>
<body>
    <h1>Original Content</h1>
    <p>Generated by JavaScript</p>
</body>
</html>
```

## Časté otázky a úskalí

| Question | Answer |
|----------|--------|
| **Mohu spouštět více skriptů sekvenčně?** | Ano. Stačí znovu zavolat `scriptEngine.execute()` s novým řetězcem skriptu. Stejná instance `HTMLDocument` si zachová svůj stav. |
| **Co když HTML soubor postrádá tag `<body>`?** | Aspose.HTML během parsování automaticky vytvoří element `<body>`, takže skript stále funguje. Pro předvídatelné výsledky však začněte s dobře formovanou šablonou. |
| **Je tento přístup kompatibilní s Java 17?** | Ano. Knihovna cílí na Java 8+, takže novější runtime jsou v pořádku. |
| **Potřebuji kompletní prohlížečové prostředí?** | Ne. Aspose.HTML poskytuje lehký, headless engine, který interpretuje standardní DOM API bez spouštění Chrome nebo Firefoxu. |
| **Jak zacházet s velkými HTML soubory?** | Načtěte dokument jako stream (`new HTMLDocument(InputStream)`) aby se předešlo načtení celého souboru najednou do paměti. |
