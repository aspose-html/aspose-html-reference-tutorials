## Vanliga frågor

**Q:** *Kan jag konvertera skyddade EPUB-filer till PDF?*  
**A:** Ja. Aspose.HTML for Java kan öppna krypterade EPUB-filer när du anger rätt dekrypteringslösenord.

**Q:** *Vad händer om jag behöver att PDF:en är PDF/A-kompatibel?*  
**A:** Använd klassen `PdfSaveOptions` för att sätta `Compliance` till `PdfCompliance.PdfA1b` (eller andra PDF/A-nivåer) innan du sparar.

**Q:** *Är det möjligt att extrahera bilder som JPG eller PNG under konverteringen?*  
**A:** Absolut. Genom att ange ett bildsparalternativ eller använda den anpassade strömleverantören kan du dirigera varje sida eller bild till JPG- eller PNG-format.

**Q:** *Behöver jag ett separat bibliotek för bildextraktion?*  
**A:** Nej. Samma Aspose.HTML for Java API hanterar både PDF-generering och bildextraktion, vilket förenklar arbetsflödet.

**Q:** *Vilka Java-versioner stöds?*  
**A:** Biblioteket stödjer Java 8 och nyare, inklusive Java 11, Java 17 och senare LTS-versioner.
