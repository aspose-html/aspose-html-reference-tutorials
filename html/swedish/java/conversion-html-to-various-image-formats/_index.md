## Hur man **convert html gif**, **convert html jpeg** och **convert html png** med Aspose.HTML

Nedan hittar du korta steg‑för‑steg‑instruktioner för varje målformat. Kodsnuttarna är identiska förutom filändelsen; du behöver bara ändra formatet i `ImageSaveOptions`.

### Steg 1: Ställ in HTML‑renderaren
Skapa en `HtmlRenderer`‑instans och läs in HTML‑källan (fil, URL eller sträng).

### Steg 2: Välj önskat bildformat
Välj en av `ImageSaveOptions`‑subklasserna – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions` osv.

### Steg 3: Rendera och spara
Anropa `renderer.renderToFile(outputPath, saveOptions)` för att generera bilden.

*De samma tre stegen gäller för BMP, TIFF och alla andra stödda format.*
