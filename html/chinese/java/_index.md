## 常见问题及解决方案
- **Missing fonts in PDF** – 确保服务器上已安装所需字体，或使用 `PdfSaveOptions` 嵌入字体。  
- **Large EPUB files cause memory pressure** – 使用基于流的处理（`MemoryStream` → `FileStream`）以降低堆内存占用。  
- **Canvas rendering appears blank** – 在调用转换 API 前，确认 Canvas 已完整绘制。
