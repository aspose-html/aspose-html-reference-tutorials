## 常见问题及解决方案
- **PDF 中缺少字体** – 确保服务器上已安装所需字体，或使用 `PdfSaveOptions` 嵌入字体。  
- **大型 EPUB 文件导致内存压力** – 使用基于流的处理（`MemoryStream` → `FileStream`）以降低堆内存使用。  
- **Canvas 渲染为空白** – 在调用转换 API 前，确认 canvas 已完整绘制。

**Q: 我如何自动填充多个 HTML 表单？**

**Q: 能否直接将 HTML 页面转换为 SVG？**
