## 常见问题

**Q:** *我可以将受保护的 EPUB 文件转换为 PDF 吗？*  
**A:** 可以。Aspose.HTML for Java 在您提供相应的解密密码后，能够打开加密的 EPUB。

**Q:** *如果我需要 PDF 符合 PDF/A 标准怎么办？*  
**A:** 在保存之前，使用 `PdfSaveOptions` 类将 `Compliance` 设置为 `PdfCompliance.PdfA1b`（或其他 PDF/A 级别）。

**Q:** *转换过程中能提取 JPG 或 PNG 等图像吗？*  
**A:** 完全可以。通过指定图像保存选项或使用自定义流提供程序，您可以将每页或每个图像导出为 JPG 或 PNG 格式。

**Q:** *我需要单独的库来提取图像吗？*  
**A:** 不需要。相同的 Aspose.HTML for Java API 同时处理 PDF 生成和图像提取，简化工作流。

**Q:** *支持哪些 Java 版本？*  
**A:** 该库支持 Java 8 及更高版本，包括 Java 11、Java 17 以及后续的 LTS 发行版。
