**问：我可以在商业应用中使用 Aspose.HTML for Java 吗？**  
答：可以。生产使用需要有效的商业许可证；提供免费试用供评估。

**问：支持哪些 Java 版本？**  
答：Aspose.HTML for Java 支持 Java 8、11 及更高版本。

**问：库是否处理外部 CSS 和 JavaScript？**  
答：当然。渲染器会处理链接的 CSS 文件并执行 JavaScript，以确保 PDF 与原始页面布局一致。

答：可以。保存前可通过 `PdfSaveOptions` 对象设置标题、作者、主题和关键字。
