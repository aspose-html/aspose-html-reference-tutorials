## How to **convert html gif**, **convert html jpeg**, and **convert html png** with Aspose.HTML

Below you’ll find concise step‑by‑step instructions for each target format. The code snippets are identical except for the output file extension; you only need to change the `ImageSaveOptions` format.

### Step 1: Set up the HTML renderer
Create an `HtmlRenderer` instance and load the HTML source (file, URL, or string).

### Step 2: Choose the desired image format
Select one of the `ImageSaveOptions` subclasses – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions`, etc.

### Step 3: Render and save
Call `renderer.renderToFile(outputPath, saveOptions)` to generate the image.

*The same three steps apply to BMP, TIFF, and any other supported format.*
