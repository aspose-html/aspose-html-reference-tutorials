# Μετατροπή EPUB σε XPS με Aspose.HTML για Java

Σε αυτό το ολοκληρωμένο tutorial, θα μάθετε πώς να **μετατρέπετε μαζικά EPUB σε XPS** χρησιμοποιώντας το Aspose.HTML για Java. Θα περάσουμε από κάθε βήμα — από τη φόρτωση ενός αρχείου EPUB σε Java μέχρι την προσαρμογή της εξόδου XPS — ώστε όχι μόνο να λειτουργεί ο κώδικας, αλλά και να κατανοείτε γιατί κάθε μέρος είναι σημαντικό.

## Γρήγορες Απαντήσεις
- **Τι καλύπτει αυτό το tutorial;** Μετατροπή ενός αρχείου EPUB σε μορφή XPS με Aspose.HTML για Java.  
- **Ποια βιβλιοθήκη απαιτείται;** Aspose.HTML για Java (εμπορική, διαθέσιμη δωρεάν δοκιμή).  
- **Χρειάζομαι ειδικό IDE;** Οποιοδήποτε Java IDE (IntelliJ, Eclipse, VS Code) που υποστηρίζει Java 8+.  
- **Μπορώ να προσαρμόσω την εμφάνιση του XPS;** Ναι — χρώμα φόντου, μέγεθος σελίδας και άλλα μέσω του `XpsSaveOptions`.  
- **Πού αποθηκεύεται η έξοδος;** Σε διαδρομή που ορίζετε, π.χ., `EPUBtoXPS_Output.xps`.

## Τι είναι η «μετατροπή epub σε xps»;
Η μετατροπή EPUB σε XPS σημαίνει τη μετατροπή ενός πακέτου ηλεκτρονικού βιβλίου (EPUB) σε έγγραφο σταθερής διάταξης (XPS) που διατηρεί τη διάταξη, τις γραμματοσειρές και τα γραφικά. Το XPS είναι χρήσιμο για εκτύπωση, αρχειοθέτηση ή προβολή σε συσκευές Windows.

## Γιατί να χρησιμοποιήσετε το Aspose.HTML για Java;
Το Aspose.HTML προσφέρει μια υψηλής απόδοσης, καθαρά Java μηχανή που διαχειρίζεται HTML, EPUB και άλλες μορφές web χωρίς εξωτερικές εξαρτήσεις. Σας παρέχει λεπτομερή έλεγχο των επιλογών μετατροπής, καθιστώντας το ιδανικό για διαδρόμους εγγράφων στο διακομιστή.

## Γιατί η μαζική μετατροπή EPUB σε XPS;
Η μαζική μετατροπή σας επιτρέπει να επεξεργαστείτε πολλά e‑books σε μία εκτέλεση — ιδανικό για ψηφιακές βιβλιοθήκες, ροές εργασίας εκδόσεων ή αυτοματοποιημένη αρχειοθέτηση. Χρησιμοποιώντας την ίδια παρουσία `XpsSaveOptions` για πολλά αρχεία, διατηρείτε την έξοδο συνεπή ενώ εξοικονομείτε χρόνο ανάπτυξης.

## Προαπαιτούμενα

- **Περιβάλλον Ανάπτυξης Java** – Εγκατεστημένο JDK 8 ή νεότερο.  
- **Aspose.HTML για Java** – Κατεβάστε τη βιβλιοθήκη από την επίσημη ιστοσελίδα και προσθέστε την στο classpath του έργου σας.  
- **Έγγραφο EPUB** – Έχετε ένα αρχείο `.epub` έτοιμο για δοκιμή της μετατροπής.

## Εισαγωγή Πακέτων

Πρώτα, εισάγετε τις κλάσεις που θα χρειαστείτε. Το παρακάτω μπλοκ κώδικα παραμένει αμετάβλητο από το αρχικό tutorial:

```java
import com.aspose.html.drawing.Color;
import com.aspose.html.saving.XpsSaveOptions;
import com.aspose.html.converters.Converter;
import java.io.FileInputStream;
```

Τώρα που έχουν εισαχθεί τα απαραίτητα πακέτα, ας περάσουμε από τα βήματα μετατροπής.

## Πώς να μετατρέψετε μαζικά EPUB σε XPS – Διαδικασία Μετατροπής

Ακολουθήστε αυτά τα αριθμημένα βήματα. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση και τον ακριβή κώδικα που χρειάζεστε.

### Βήμα 1: Φόρτωση του Εγγράφου EPUB σε Java

Η φόρτωση ενός αρχείου EPUB είναι τόσο απλή όσο το άνοιγμα ενός `FileInputStream`. Εδώ εμφανίζεται φυσικά η δευτερεύουσα λέξη-κλειδί **load epub java**.

```java
try (FileInputStream fileInputStream = new FileInputStream("input.epub")) {
    // Your code here
}
```

> **Συμβουλή:** Τυλίξτε το `FileInputStream` σε ένα μπλοκ try‑with‑resources (όπως φαίνεται) για να εξασφαλίσετε ότι η ροή κλείνει αυτόματα.

### Βήμα 2: Αρχικοποίηση του `XpsSaveOptions`

`XpsSaveOptions` σας επιτρέπει να ρυθμίσετε την έξοδο XPS. Σε αυτό το παράδειγμα ορίζουμε ένα κυανό φόντο, αλλά μπορείτε να προσαρμόσετε οποιαδήποτε ιδιότητα χρειάζεστε.

```java
XpsSaveOptions options = new XpsSaveOptions();
options.setBackgroundColor(Color.getCyan());
```

### Βήμα 3: Καθορισμός της Διαδρομής Αρχείου Εξόδου

Αποφασίστε πού θα αποθηκευτεί το μετατρεπόμενο αρχείο XPS. Αλλάξτε το όνομα αρχείου ή τον φάκελο όπως απαιτείται.

```java
String outputFile = "EPUBtoXPS_Output.xps";
```

### Βήμα 4: Εκτέλεση της Μετατροπής

Τέλος, καλέστε το `Converter.convertEPUB` με τη ροή εισόδου, τις επιλογές και τη διαδρομή εξόδου.

```java
Converter.convertEPUB(fileInputStream, options, outputFile);
```

Όταν εκτελεστεί αυτή η γραμμή, το Aspose.HTML διαβάζει το EPUB, εφαρμόζει τις επιλογές XPS και γράφει το αποτέλεσμα στο `EPUBtoXPS_Output.xps`.

## Συχνά Προβλήματα και Λύσεις

| Πρόβλημα | Αιτία | Διόρθωση |
|----------|-------|----------|
| **`FileNotFoundException`** | Λάθος διαδρομή προς το `input.epub` | Επαληθεύστε ότι το αρχείο υπάρχει σχετικά με τον τρέχοντα φάκελο ή χρησιμοποιήστε απόλυτη διαδρομή. |
| **Missing fonts in XPS** | Οι γραμματοσειρές δεν είναι ενσωματωμένες στο EPUB | Βεβαιωθείτε ότι το EPUB περιλαμβάνει τις απαιτούμενες γραμματοσειρές ή εγκαταστήστε τις στο σύστημα. |
| **Out‑of‑memory error** | Πολύ μεγάλα αρχεία EPUB | Αυξήστε το μέγεθος heap της JVM (`-Xmx2g`) ή επεξεργαστείτε το EPUB σε μικρότερα τμήματα αν είναι δυνατόν. |

## Συχνές Ερωτήσεις

**Q: Μπορώ να μετατρέψω αρχεία EPUB με κωδικό πρόσβασης;**  
A: Ναι. Ανοίξτε το EPUB με ένα `FileInputStream` αφού παρέχετε τον κωδικό πρόσβασης στη βασική ροή, και στη συνέχεια περάστε το στο `Converter.convertEPUB`.

**Q: Πώς αλλάζω το μέγεθος σελίδας του παραγόμενου XPS;**  
A: Ορίστε `options.setPageSize(width, height)` όπου το πλάτος και το ύψος είναι σε points.

**Q: Είναι δυνατόν να μετατρέψω πολλά αρχεία EPUB σε μια παρτίδα;**  
A: Απόλυτα. Επαναλάβετε (loop) πάνω σε μια λίστα διαδρομών αρχείων, χρησιμοποιώντας ξανά την ίδια παρουσία `XpsSaveOptions` για κάθε μετατροπή.

**Q: Υποστηρίζει το Aspose.HTML εικόνες SVG μέσα σε EPUB;**  
A: Ναι. Το περιεχόμενο SVG αποδίδεται σωστά κατά τη μετατροπή σε XPS.
