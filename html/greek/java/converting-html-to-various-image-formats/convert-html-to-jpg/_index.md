## Εισαγωγή Πακέτων

Το πρώτο βήμα είναι να φέρετε τις απαιτούμενες κλάσεις Aspose.HTML στο έργο σας:

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
import java.io.FileWriter;
```

## Προετοιμασία του Κώδικα HTML (αποθήκευση html ως jpg)

Δημιουργήστε ένα ελάχιστο απόσπασμα HTML ή δείξτε σε υπάρχον αρχείο. Σε αυτό το παράδειγμα γράφουμε ένα μικρό αρχείο HTML εν κινήσει:

```java
String code = "<span>Hello</span> <span>World!!</span>";
try (FileWriter fileWriter = new FileWriter("document.html")) {
    fileWriter.write(code);
}
```

> **Συμβουλή:** Για μεγαλύτερες σελίδες, φορτώστε το HTML από URL ή ροή πόρων αντί να γράψετε ένα προσωρινό αρχείο.

## Αρχικοποίηση του Εγγράφου HTML

Φορτώστε το αρχείο HTML σε ένα αντικείμενο `HTMLDocument`, το οποίο το Aspose.HTML θα αποδώσει αργότερα:

```java
HTMLDocument document = new HTMLDocument("document.html");
```

## Διαμόρφωση ImageSaveOptions (δημιουργία jpg από html)

Ορίστε τη μορφή εξόδου σε JPEG και προαιρετικά προσαρμόστε την ποιότητα ή τις διαστάσεις:

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Jpeg);
```

Μπορείτε επίσης να ορίσετε `options.setQuality(90);` για να ελέγξετε τη συμπίεση.

## Μετατροπή HTML σε JPG

Με το έγγραφο και τις επιλογές έτοιμες, καλέστε τον μετατροπέα για να δημιουργήσετε την εικόνα:

```java
Converter.convertHTML(document, options, "output.jpg");
```

Αυτή η μοναδική γραμμή εκτελεί ολόκληρη τη διαδικασία μετατροπής **java html to image**.

## Java HTML σε Εικόνα Εκπαίδευση – Πρόσθετες Συμβουλές

- **Απόδοση HTML ως JPEG με προσαρμοσμένες διαστάσεις:** Ρυθμίστε `options.setWidth()` και `options.setHeight()` για να ελέγξετε το μέγεθος εξόδου.  
- **Μαζική μετατροπή:** Επανάληψη σε φάκελο αρχείων HTML και κλήση `Converter.convertHTML` για κάθε αρχείο, επαναχρησιμοποιώντας το ίδιο αντικείμενο `ImageSaveOptions` για βελτιωμένη απόδοση.  
- **Ενσωμάτωση γραμματοσειρών:** Εάν το HTML σας χρησιμοποιεί προσαρμοσμένες γραμματοσειρές, βεβαιωθείτε ότι είναι διαθέσιμες στο classpath ή ενσωματωμένες μέσω κανόνων `@font-face`; διαφορετικά η παραγόμενη εικόνα μπορεί να επιστρέψει στις προεπιλεγμένες γραμματοσειρές.
