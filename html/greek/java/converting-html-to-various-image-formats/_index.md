Η δημιουργία μιας εικόνας PNG από ένα αρχείο HTML σημαίνει την απόδοση του κώδικα HTML, του στυλ CSS και τυχόν ενσωματωμένων πόρων (γραμματοσειρές, εικόνες, SVG) σε ένα ραστερ bitmap που μπορεί να αποθηκευτεί ως αρχείο PNG. Αυτό είναι χρήσιμο όταν χρειάζεστε μια στατική λήψη δυναμικού περιεχομένου ιστού για αναφορές, μικρογραφίες email ή εκτός σύνδεσης τεκμηρίωση.

## Μετατροπή HTML σε BMP

Όταν χρειάζεστε **convert html to bmp**, η κλάση `ImageSaveOptions` του Aspose.HTML σας επιτρέπει να ορίσετε το BMP ως μορφή εξόδου. Η διαδικασία είναι απλή:

1. Φορτώστε το έγγραφο HTML χρησιμοποιώντας το `HTMLDocument`.  
2. Δημιουργήστε ένα αντικείμενο `ImageSaveOptions` και ορίστε το `ImageFormat` σε BMP.  
3. Καλέστε τη μέθοδο `save` με το επιθυμητό όνομα αρχείου και το αντικείμενο επιλογών.

> *Pro tip:* Τα αρχεία BMP είναι μεγάλα επειδή δεν είναι συμπιεσμένα. Χρησιμοποιήστε τα μόνο όταν η απώλεια ποιότητας είναι αυστηρή απαίτηση.

## Μετατροπή HTML σε GIF

Η ροή εργασίας **convert html to gif** είναι παρόμοια, αλλά μπορείτε επίσης να ενεργοποιήσετε την υποστήριξη κίνησης εάν το HTML σας περιέχει κινούμενα στοιχεία (π.χ., CSS animations). Ορίστε το `ImageFormat` σε GIF και, αν χρειάζεται, προσαρμόστε τις επιλογές καθυστέρησης πλαισίου.

Το GIF είναι ιδανικό για μικρές κινήσεις ή όταν χρειάζεστε περιορισμένη παλέτα χρωμάτων (μέγιστο 256 χρώματα).  

## Μετατροπή HTML σε JPG

Για το σενάριο **convert html to jpg**, επωφελείστε από μικρότερα μεγέθη αρχείων χάρη στη μεθόδους συμπίεσης με απώλειες του JPEG. Αυτή η μορφή λειτουργεί καλύτερα για φωτογραφικό περιεχόμενο όπου μια μικρή απώλεια λεπτομέρειας είναι αποδεκτή.

Θυμηθείτε να ορίσετε την ιδιότητα `Quality` στο `JpegOptions` εάν χρειάζεστε πιο ακριβή έλεγχο των επιπέδων συμπίεσης.

## Μετατροπή HTML σε PNG

Αν ο στόχος σας είναι **create png from html**, το PNG προσφέρει συμπίεση χωρίς απώλειες και υποστηρίζει διαφάνεια—ιδανικό για λογότυπα, στοιχεία UI ή οποιαδήποτε γραφικά που απαιτούν καθαρό φόντο.

Ορίστε το `ImageFormat` σε PNG και προαιρετικά καθορίστε το `Resolution` για βελτίωση της ευκρίνειας σε οθόνες υψηλής ανάλυσης (high‑DPI).
