## Μετατροπή HTML σε BMP

Όταν χρειάζεστε μια ροή εργασίας **convert html to bmp**, το BMP παρέχει μια ασυμπίεστη, υψηλής ποιότητας ραστερ εικόνα που είναι ιδανική για περαιτέρω επεξεργασία εικόνας. Ακολουθήστε αυτά τα βήματα:

1. Φορτώστε την πηγή HTML χρησιμοποιώντας `HtmlDocument`.  
2. Δημιουργήστε ένα αντικείμενο `ImageSaveOptions` που καθορίζει το `Bmp` ως μορφή εξόδου.  
3. Καλέστε `document.save("output.bmp", options)`.

> *Pro tip:* Τα αρχεία BMP είναι μεγαλύτερα· χρησιμοποιήστε τα όταν χρειάζεστε δεδομένα χωρίς απώλειες για επεξεργασία σε επόμενο στάδιο.

## Μετατροπή HTML σε GIF

Αν θέλετε να **convert html to gif**, ειδικά για απλές κινήσεις, το Aspose.HTML μπορεί να αποδώσει κάθε καρέ και να τα συναρμολογήσει σε ένα animated GIF:

1. Αποδώστε κάθε κατάσταση HTML (π.χ., διαφορετικές κλάσεις CSS) σε ξεχωριστές εικόνες.  
2. Χρησιμοποιήστε `GifSaveOptions` για να συνδυάσετε τα καρέ, ορίζοντας την καθυστέρηση του καρέ όπως απαιτείται.  
3. Αποθηκεύστε το αποτέλεσμα με `document.save("animation.gif", options)`.

> *Why GIF?* Είναι ιδανικό για μικρές, επαναλαμβανόμενες κινήσεις ή όταν χρειάζεστε ευρεία συμβατότητα σε όλα τα προγράμματα περιήγησης.

## Μετατροπή HTML σε JPG

Ακολουθεί το βασικό παράδειγμα **java html to jpg**. Το JPG είναι η προτιμώμενη μορφή για φωτογραφίες και εικόνες έτοιμες για web όπου το μέγεθος αρχείου μετρά:

1. Φορτώστε το HTML με `HtmlDocument`.  
2. Προετοιμάστε `JpegSaveOptions`, προαιρετικά ρυθμίζοντας την ποιότητα (0‑100).  
3. Αποθηκεύστε το αποτέλεσμα: `document.save("page.jpg", options)`.

> *Pro tip:* Ορίστε την ποιότητα JPEG στο 80 % για καλή ισορροπία μεταξύ οπτικής πιστότητας και μεγέθους αρχείου.

## Μετατροπή HTML σε PNG

Το PNG σας παρέχει ασυμπίεστη συμπίεση και υποστηρίζει διαφάνεια—ιδανικό για στοιχεία UI και λογότυπα. Για **convert html to png**:

1. Φορτώστε το έγγραφο HTML.  
2. Χρησιμοποιήστε `PngSaveOptions`; μπορείτε να ενεργοποιήσετε το `Transparency` αν χρειάζεται.  
3. Αποθηκεύστε: `document.save("image.png", options)`.

> *When to choose PNG?* Όποτε χρειάζεστε καθαρά άκρα, κανάλια άλφα, ή όταν η εικόνα θα υποστεί επεξεργασία αργότερα.
