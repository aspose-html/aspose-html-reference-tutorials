## Πώς να εκτελέσετε πώς να μετατρέψετε epub σε pdf

Ακολουθεί ένας βήμα‑βήμα οδηγός. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση, ακολουθούμενη από το αρχικό τμήμα κώδικα (αμετάβλητο).

### Βήμα 1: Άνοιγμα του Πηγαίου Εγγράφου EPUB

Ξεκινάμε ανοίγοντας το πηγαίο αρχείο EPUB με χρήση `FileInputStream`. Αυτό το ρεύμα τροφοδοτεί τον μετατροπέα με τα ακατέργαστα δεδομένα EPUB.

```java
try (java.io.FileInputStream fileInputStream = new java.io.FileInputStream("input.epub")) {
```

### Βήμα 2: Αρχικοποίηση `PdfSaveOptions`

Δημιουργήστε μια παρουσία του `PdfSaveOptions` για να ρυθμίσετε τη μετατροπή PDF. Για παράδειγμα, μπορείτε να ορίσετε την ποιότητα JPEG ώστε οι εικόνες να διατηρούν υψηλή πιστότητα.

```java
PdfSaveOptions options = new PdfSaveOptions();
options.setJpegQuality(100);
```

### Βήμα 3: Ορισμός Διαδρομής Αρχείου Εξόδου

Καθορίστε πού θα αποθηκευτεί το παραγόμενο PDF. Μπορείτε να αλλάξετε το όνομα αρχείου ή τον φάκελο όπως απαιτείται.

```java
String outputFile = "EPUBtoPDF_Output.pdf";
```

### Βήμα 4: Μετατροπή EPUB σε PDF

Τέλος, καλέστε τη μέθοδο `convertEPUB`, περνώντας το ρεύμα εισόδου, τις επιλογές αποθήκευσης και τη διαδρομή εξόδου.
