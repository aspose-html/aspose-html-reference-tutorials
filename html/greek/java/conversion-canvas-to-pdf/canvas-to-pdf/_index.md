## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Φόρτωση του HTML Εγγράφου
```java
com.aspose.html.HTMLDocument document = new com.aspose.html.HTMLDocument(Resources.input("canvas.html"));
```
Εδώ φορτώνουμε το πηγαίο HTML που περιλαμβάνει το canvas. Αντικαταστήστε το `"canvas.html"` με τη διαδρομή του δικού σας αρχείου.

### Βήμα 2: Δημιουργία ενός HTML Renderer
```java
com.aspose.html.rendering.HtmlRenderer renderer = new com.aspose.html.rendering.HtmlRenderer();
```
Ο renderer είναι υπεύθυνος για τη μετατροπή του HTML (συμπεριλαμβανομένου του canvas) σε οπτική αναπαράσταση που μπορεί να γραφτεί σε μια συσκευή PDF.

### Βήμα 3: Αρχικοποίηση της Συσκευής PDF
```java
com.aspose.html.rendering.pdf.PdfDevice device = new com.aspose.html.rendering.pdf.PdfDevice(Resources.output("canvas.output.pdf"));
```
Το `PdfDevice` ορίζει πού θα αποθηκευτεί η παραγόμενη έξοδος. Αλλάξτε το `"canvas.output.pdf"` στο επιθυμητό όνομα αρχείου εξόδου.

### Βήμα 4: Απόδοση του Εγγράφου
```java
renderer.render(device, document);
```
Αυτή η μοναδική γραμμή εκτελεί την κύρια λειτουργία **convert canvas to pdf java**. Ο renderer διαβάζει το HTML, ζωγραφίζει το canvas και μεταβιβάζει το αποτέλεσμα στη συσκευή PDF.

### Βήμα 5: Καθαρισμός Πόρων
```java
device.dispose();
renderer.dispose();
document.dispose();
```
Η απελευθέρωση των αντικειμένων αποδεσμεύει τους εγγενείς πόρους και αποτρέπει διαρροές μνήμης—ιδιαίτερα σημαντικό όταν επεξεργάζεστε πολλά αρχεία σε batch job.
