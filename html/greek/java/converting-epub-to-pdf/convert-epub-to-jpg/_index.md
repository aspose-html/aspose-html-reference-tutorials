## Εισαγωγή Πακέτων

Για να ξεκινήσετε, πρέπει να εισάγετε τα απαραίτητα πακέτα από το Aspose.HTML για Java. Αυτές οι εισαγωγές σας δίνουν πρόσβαση στη μηχανή μετατροπής και στις επιλογές αποθήκευσης εικόνας.

```java
// Import the required Aspose.HTML for Java packages
import com.aspose.html.saving.ImageSaveOptions;

import java.io.FileInputStream;
```

## Οδηγός Βήμα‑βήμα

### Βήμα 1: Άνοιγμα του αρχείου EPUB
Πρώτα, ανοίγουμε το αρχείο EPUB χρησιμοποιώντας ένα `FileInputStream`. Αντικαταστήστε το `'input.epub'` με την πραγματική διαδρομή του εγγράφου EPUB.

```java
try (FileInputStream fileInputStream = new FileInputStream("input.epub")) {
    // Your code for the next steps will go here.
}
```

> **Συμβουλή:** Η χρήση ενός μπλοκ try‑with‑resources εξασφαλίζει ότι η ροή κλείνει αυτόματα, αποτρέποντας διαρροές χειριστών αρχείων.

### Βήμα 2: Αρχικοποίηση ImageSaveOptions  
Στη συνέχεια, δημιουργούμε ένα αντικείμενο `ImageSaveOptions` για να ορίσουμε πώς θα αποθηκευτούν οι εικόνες εξόδου. Εδώ επιλέγουμε JPEG ως μορφή στόχο, που είναι ιδανική για μικρογραφίες έτοιμες για το web.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Jpeg);
```

Μπορείτε επίσης να ρυθμίσετε την ποιότητα, το DPI και τις διαστάσεις σελίδας μέσω του αντικειμένου `options` εάν χρειάζεστε πιο ακριβή έλεγχο.

### Βήμα 3: Μετατροπή EPUB σε JPG  
Τέλος, καλέστε τη μέθοδο μετατροπής. Το `Converter.convertEPUB` διαβάζει τη ροή EPUB, αποδίδει κάθε σελίδα και γράφει το αποτέλεσμα στο καθορισμένο αρχείο JPG.

```java
Converter.convertEPUB(fileInputStream, options, "output.jpg");
```

> **Συνηθισμένο λάθος:** Βεβαιωθείτε ότι η διαδρομή του αρχείου εξόδου τελειώνει με “.jpg”. Εάν χρειάζεστε ξεχωριστές εικόνες ανά σελίδα, δώστε ένα μοτίβο όπως `"output_page_{0}.jpg"`.
