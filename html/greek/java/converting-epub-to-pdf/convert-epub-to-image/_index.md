## Προαπαιτούμενα

1. **Java Development Kit (JDK)** – Java 8 ή νεότερο εγκατεστημένο στο σύστημά σας.  
2. **Aspose HTML for Java library** – κατεβάστε το από [the website](https://releases.aspose.com/html/java/).  
3. **An EPUB file** που θέλετε να μετατρέψετε – βεβαιωθείτε ότι το αρχείο είναι προσβάσιμο από το πρόγραμμα Java σας.

## Εισαγωγή Πακέτων

Πριν γράψετε οποιονδήποτε κώδικα μετατροπής, εισάγετε τις απαιτούμενες κλάσεις:

```java
import java.io.FileInputStream;
import com.aspose.html.converters.Converter;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Άνοιγμα του αρχείου EPUB
Δημιουργήστε ένα `FileInputStream` που δείχνει στο πηγαίο σας EPUB. Το μπλοκ `try‑with‑resources` εξασφαλίζει ότι η ροή κλείνει αυτόματα.

```java
try (FileInputStream fileInputStream = new FileInputStream("path/to/your/input.epub")) {
    // The conversion code will be placed here.
}
```

Αντικαταστήστε `"path/to/your/input.epub"` με το πραγματικό μονοπάτι του συστήματος αρχείων του EPUB σας.

### Βήμα 2: Μετατροπή EPUB σε Εικόνα
Μέσα στο μπλοκ `try`, καλέστε το `Converter.convertEPUB`. Παρέχετε τη ροή εισόδου, ένα αντικείμενο `ImageSaveOptions` που ορίζει τη μορφή εξόδου, και το μονοπάτι του αρχείου προορισμού.

```java
Converter.convertEPUB(
    fileInputStream,
    new ImageSaveOptions(ImageFormat.Jpeg),
    "path/to/your/output.jpg"
);
```

Αν προτιμάτε PNG, απλώς αλλάξτε το `ImageFormat.Jpeg` σε `ImageFormat.Png` και προσαρμόστε την επέκταση του αρχείου εξόδου ανάλογα.

### Βήμα 3: Επαλήθευση της Εξόδου
Αφού ολοκληρωθεί η μετατροπή, μεταβείτε στο φάκελο εξόδου που καθορίσατε. Θα πρέπει να δείτε μια εικόνα υψηλής ποιότητας που αντικατοπτρίζει την αρχική διάταξη του EPUB. Ανοίξτε την με οποιονδήποτε προβολέα εικόνων για να το επιβεβαιώσετε.
