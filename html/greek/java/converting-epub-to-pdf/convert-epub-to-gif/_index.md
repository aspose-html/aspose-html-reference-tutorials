## Εισαγωγή Πακέτων

Προσθέστε τις απαραίτητες εισαγωγές Aspose.HTML στο έργο Java σας:

```java
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.converters.Converter;
import java.io.FileInputStream;
```

## Οδηγός Βήμα‑Βήμα

### Βήμα 1: Άνοιγμα του αρχείου EPUB
Πρώτα, ανοίξτε το πηγαίο EPUB χρησιμοποιώντας ένα `FileInputStream`. Αυτό το ρεύμα τροφοδοτεί τον μετατροπέα με τα ακατέργαστα δεδομένα του EPUB.

```java
try (FileInputStream fileInputStream = new FileInputStream("input.epub")) {
    // Your code for conversion goes here.
}
```

Αντικαταστήστε το `"input.epub"` με την πραγματική διαδρομή του αρχείου EPUB σας.

### Βήμα 2: Αρχικοποίηση ImageSaveOptions
Διαμορφώστε τη μορφή εξόδου δημιουργώντας μια παρουσία `ImageSaveOptions` και ορίζοντας το GIF ως μορφή προορισμού.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Gif);
```

Αυτό λέει στο Aspose.HTML να αποδώσει τις σελίδες του EPUB ως εικόνες GIF.

### Βήμα 3: Μετατροπή EPUB σε GIF
Τέλος, καλέστε τον μετατροπέα για να δημιουργήσετε το αρχείο GIF. Η μέθοδος `convertEPUB` διαβάζει από το ρεύμα εισόδου, εφαρμόζει τις επιλογές και γράφει το αποτέλεσμα στη διαδρομή εξόδου που καθορίζεται.

```java
Converter.convertEPUB(fileInputStream, options, "output.gif");
```

Μετά την εκτέλεση, θα βρείτε το `output.gif` στον φάκελο του έργου σας, έτοιμο για χρήση.

## Συχνά Προβλήματα και Λύσεις
- **FileNotFoundException** – Επαληθεύστε ότι η διαδρομή του EPUB είναι σωστή και το αρχείο είναι προσβάσιμο.  
- **OutOfMemoryError** – Για μεγάλα EPUB, αυξήστε το μέγεθος heap της JVM (`-Xmx` flag) ή επεξεργαστείτε τις σελίδες ξεχωριστά.  
- **Missing fonts** – Βεβαιωθείτε ότι οι γραμματοσειρές που χρησιμοποιούνται στο EPUB είναι εγκατεστημένες στο σύστημα ή ενσωματωμένες στο EPUB.
