# Δημιουργία PDF από EPUB χρησιμοποιώντας Aspose.HTML για Java

## Εισαγωγή

Στη σύγχρονη ανάπτυξη Java, η **δημιουργία PDF από EPUB** είναι μια κοινή απαίτηση όταν χρειάζεται να διανείμετε ηλεκτρονικά βιβλία σε συσκευές που καταλαβαίνουν μόνο PDF. Το Aspose.HTML για Java σας παρέχει έναν απλό, υψηλής απόδοσης τρόπο για να εκτελέσετε αυτή τη μετατροπή *epub σε pdf* χωρίς να αφήσετε το οικοσύστημα Java. Σε αυτόν τον οδηγό θα περάσουμε από ένα πλήρες **epub to pdf tutorial**, δείχνοντάς σας ακριβώς πώς να μετατρέψετε ένα αρχείο EPUB σε έγγραφο PDF με μόνο λίγες γραμμές κώδικα.

## Γρήγορες Απαντήσεις
- **Τι κάνει η βιβλιοθήκη;** Το Aspose.HTML για Java μετατρέπει EPUB, HTML και άλλες μορφές web απευθείας σε PDF.  
- **Πόσες γραμμές κώδικα απαιτούνται;** Περίπου 5‑6 γραμμές (εισαγωγές, φόρτωση, επιλογές, μετατροπή).  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Ποια έκδοση της Java υποστηρίζεται;** Java 8 και νεότερες.  
- **Μπορώ να προσαρμόσω την έξοδο PDF;** Ναι – χρησιμοποιήστε `PdfSaveOptions` για να ορίσετε το μέγεθος σελίδας, τα περιθώρια και άλλα.

Ανοίξτε το έργο Java και εισάγετε τα απαραίτητα πακέτα του Aspose.HTML για Java. Αυτές οι εισαγωγές σας δίνουν πρόσβαση στις κλάσεις μετατροπέα και επιλογών PDF.

## Φόρτωση του Αρχείου EPUB

Στη συνέχεια, φορτώστε το EPUB που θέλετε να μετατρέψετε. Αντικαταστήστε το `"input.epub"` με τη διαδρομή του πραγματικού σας αρχείου.

```java
try (FileInputStream fileInputStream = new FileInputStream("input.epub")) {
```

Αυτό το απόσπασμα ανοίγει το EPUB για ανάγνωση και εξασφαλίζει ότι η ροή κλείνει αυτόματα.

## Ορισμός Επιλογών Μετατροπής PDF

Δημιουργήστε μια παρουσία `PdfSaveOptions` για να ελέγξετε την εμφάνιση του παραγόμενου PDF (μέγεθος σελίδας, περιθώρια κ.λπ.).

```java
PdfSaveOptions options = new PdfSaveOptions();
```

Ανακαλύψτε επιπλέον ιδιότητες στο `options` για προχωρημένη προσαρμογή.

## Μετατροπή EPUB σε PDF

Τώρα εκτελέστε τη μετατροπή. Η μέθοδος `Converter.convertEPUB` διαβάζει τη ροή EPUB, εφαρμόζει τις επιλογές και γράφει το αποτέλεσμα στο `output.pdf`.

```java
Converter.convertEPUB(fileInputStream, options, "output.pdf");
```

Αφού εκτελεστεί αυτή η γραμμή, θα βρείτε το `output.pdf` στον φάκελο του έργου, έτοιμο για διανομή.
