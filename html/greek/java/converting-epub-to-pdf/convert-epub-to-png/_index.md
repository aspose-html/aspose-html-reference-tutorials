## Εισαγωγή Πακέτων
Το namespace `com.aspose.html` περιέχει όλες τις κλάσεις που θα χρειαστείτε για την απόδοση EPUB.
Η κλάση `Document` αντιπροσωπεύει ένα φορτωμένο EPUB, ενώ η `ImageSaveOptions` σας επιτρέπει να ορίσετε ρυθμίσεις ειδικές για PNG.

**Definition anchor:** `Document` είναι το βασικό αντικείμενο του Aspose.HTML που φορτώνει και αντιπροσωπεύει ένα αρχείο EPUB στη μνήμη.  

```java
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.converters.Converter;
import com.aspose.html.rendering.image.ImageFormat;
import java.io.FileInputStream;
```

## Βήμα 1: Άνοιγμα του αρχείου EPUB
Φορτώστε το EPUB σε μια παρουσίαση `Document`. Αυτό το βήμα αναλύει το container, εξάγει το περιεχόμενο HTML και το προετοιμάζει για απόδοση.

**Definition anchor:** `Converter` είναι η βοηθητική κλάση που εκτελεί μετατροπές μορφών όπως EPUB → PNG.  

```java
try (FileInputStream fileInputStream = new FileInputStream("path/to/your/input.epub")) {
```

Αντικαταστήστε το `"path/to/your/input.epub"` με την πραγματική διαδρομή προς το αρχείο EPUB σας.

## Βήμα 2: Αρχικοποίηση ImageSaveOptions
`ImageSaveOptions` σας επιτρέπει να καθορίσετε τη μορφή εξόδου, την ανάλυση και άλλες παραμέτρους σχετικές με την εικόνα. Ορίζοντας το `format` σε `ImageFormat.Png` εξασφαλίζει έξοδο PNG.

**Definition anchor:** `ImageSaveOptions` διαμορφώνει τις επιλογές εξόδου ραστερ‑εικόνας όπως μορφή, DPI και βάθος χρώματος.  

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Png);
```

## Βήμα 3: Μετατροπή EPUB σε PNG
Τώρα καλέστε τη μετατροπή. Η μέθοδος `Converter.convertEPUB` επεξεργάζεται κάθε σελίδα του EPUB και γράφει ένα αρχείο PNG για κάθε αποδομένη σελίδα.

**Definition anchor:** `convertEPUB` είναι η στατική μέθοδος που εκτελεί την πραγματική απόδοση και δημιουργία αρχείου.  

```java
Converter.convertEPUB(
    fileInputStream,
    options,
    "path/to/your/output.png"
);
```

Αντικαταστήστε το `"path/to/your/output.png"` με τον επιθυμητό φάκελο εξόδου και το μοτίβο ονόματος αρχείου. Η μέθοδος θα προσθέσει αυτόματα αριθμούς σελίδων εάν το EPUB περιέχει πολλαπλές σελίδες.

## Συχνά Προβλήματα και Λύσεις
- **File not found error:** Επαληθεύστε ότι οι διαδρομές αρχείων είναι απόλυτες ή σωστά σχετικές με τον φάκελο εργασίας του έργου σας.  
- **Out of memory for large EPUBs:** Επεξεργαστείτε το EPUB σελίδα‑με‑σελίδα (η βιβλιοθήκη διαβάζει τις σελίδες σε ροή) ή αυξήστε το μέγεθος του heap της JVM με τη σημαία `-Xmx`.  
- **Missing fonts:** Βεβαιωθείτε ότι οι γραμματοσειρές που χρησιμοποιούνται στο EPUB είναι εγκατεστημένες στον διακομιστή ή ενσωματώστε τις μέσω `FontSettings`.  
- **Incorrect image colors:** Ορίστε το `ImageSaveOptions` `colorMode` σε `ColorMode.Rgb` για συνεπή απόδοση χρωμάτων.
