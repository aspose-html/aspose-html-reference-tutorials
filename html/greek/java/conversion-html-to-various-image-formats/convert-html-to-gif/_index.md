## Μετατροπή HTML σε GIF

Παρακάτω είναι η πλήρης, εκτελέσιμη ροή. Κάθε βήμα εξηγείται με απλή γλώσσα ώστε να το προσαρμόσετε στο δικό σας έργο.

### Βήμα 1: Φόρτωση του Εγγράφου HTML
`HTMLDocument` είναι το κορυφαίο αντικείμενο του Aspose.HTML που αντιπροσωπεύει ένα μοναδικό αρχείο HTML στη μνήμη. Αναλύει τη σήμανση, επιλύει τους πόρους και προετοιμάζει τη μηχανή διάταξης.

Δημιουργήστε μια παρουσία `HTMLDocument` που δείχνει στο αρχείο προέλευσης σας.

```java
HTMLDocument htmlDocument = new HTMLDocument("your_input.html");
```

> **Pro tip:** Αν το HTML σας αναφέρεται σε εξωτερικούς πόρους (CSS, εικόνες), τοποθετήστε τα στον ίδιο φάκελο ή δώστε απόλυτο URL ώστε ο renderer να τα επιλύσει σωστά.

### Βήμα 2: Ορισμός Μορφής Εξόδου
`ImageSaveOptions` ορίζει πώς αποθηκεύεται η αποδοθείσα σελίδα ως εικόνα. Ορίζοντας την ιδιότητα `format` σε `ImageFormat.Gif`, λέτε στο Aspose.HTML να δημιουργήσει αρχείο GIF.

Ρυθμίστε το `ImageSaveOptions` ώστε να ενημερώσετε το Aspose.HTML ότι η μορφή προορισμού είναι GIF.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Gif);
```

Μπορείτε επίσης να ρυθμίσετε ιδιότητες όπως η ποιότητα εικόνας, το χρώμα φόντου ή η καθυστέρηση καρέ εδώ αν χρειάζεστε κινούμενα GIF.

### Βήμα 3: Ορισμός Διαδρομής Αρχείου Εξόδου
Καθορίστε πού θα αποθηκευτεί το παραγόμενο GIF. Η διαδρομή μπορεί να είναι απόλυτη ή σχετική με τον κατάλογο εργασίας του έργου σας.

Ορίστε τη διαδρομή αρχείου εξόδου για την εικόνα GIF.

```java
String outputFile = "output.gif";
```

### Βήμα 4: Εκτέλεση της Μετατροπής
`Converter.convertHTML` είναι η μοναδική μέθοδος που αποδίδει το έγγραφο HTML και γράφει το αρχείο εικόνας βάσει των παρεχόμενων επιλογών. Διαχειρίζεται τη φόρτωση πόρων, τον υπολογισμό διάταξης και το rasterization εσωτερικά.

Η μέθοδος `Converter.convertHTML` εκτελεί όλη τη βαριά δουλειά.

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

Μετά την εκτέλεση αυτής της γραμμής, το `output.gif` θα περιέχει ένα rasterized στιγμιότυπο της αρχικής σελίδας HTML.
