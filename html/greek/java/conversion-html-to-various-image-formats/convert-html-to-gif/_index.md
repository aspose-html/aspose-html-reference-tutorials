## Εισαγωγή Πακέτων

Οι παρακάτω εισαγωγές σας δίνουν πρόσβαση στις βασικές κλάσεις μετατροπής.

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
```

## Μετατροπή HTML σε GIF

Παρακάτω βρίσκεται η πλήρης, εκτελέσιμη ροή. Κάθε βήμα εξηγείται με απλή γλώσσα ώστε να το προσαρμόσετε στο δικό σας έργο.

### Βήμα 1: Φόρτωση του Εγγράφου HTML
Δημιουργήστε ένα αντικείμενο `HTMLDocument` που δείχνει στο αρχείο προέλευσης.

```java
HTMLDocument htmlDocument = new HTMLDocument("your_input.html");
```

> **Συμβουλή:** Αν το HTML σας αναφέρεται σε εξωτερικούς πόρους (CSS, εικόνες), τοποθετήστε τα στον ίδιο φάκελο ή δώστε απόλυτο URL ώστε ο renderer να μπορεί να τα επιλύσει σωστά.

### Βήμα 2: Ορισμός της Μορφής Εξόδου
Ρυθμίστε το `ImageSaveOptions` για να ενημερώσετε το Aspose.HTML ότι η μορφή προορισμού είναι GIF.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Gif);
```

Μπορείτε επίσης να τροποποιήσετε ιδιότητες όπως η ποιότητα εικόνας, το χρώμα φόντου ή η καθυστέρηση καρέ εδώ εάν χρειάζεστε κινούμενα GIF.

### Βήμα 3: Καθορισμός της Διαδρομής Αρχείου Εξόδου
Ορίστε πού θα αποθηκευτεί το παραγόμενο GIF.

```java
String outputFile = "output.gif";
```

### Βήμα 4: Εκτέλεση της Μετατροπής
Η μέθοδος `Converter.convertHTML` κάνει όλη τη βαριά δουλειά.

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

Μετά την εκτέλεση αυτής της γραμμής, το `output.gif` θα περιέχει ένα ραστερισμένο στιγμιότυπο της αρχικής σελίδας HTML.

- **Μεγάλες σελίδες HTML** – Αυξήστε την κατανομή μνήμης για το JVM (`-Xmx`) εάν αντιμετωπίσετε `OutOfMemoryError`.
