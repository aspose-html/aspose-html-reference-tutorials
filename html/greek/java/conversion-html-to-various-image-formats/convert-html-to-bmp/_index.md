## Τι είναι το convert html to bmp;
`convert html to bmp` είναι η διαδικασία απόδοσης ενός εγγράφου HTML σε εικόνα bitmap (BMP). Η μετατροπή καταγράφει την ακριβή οπτική διάταξη, συμπεριλαμβανομένου του κειμένου, των χρωμάτων, των γραμματοσειρών και των διανυσματικών γραφικών, και το αποθηκεύει ως αρχείο raster εικόνας. Αυτό επιτρέπει το περιεχόμενο HTML να εμφανίζεται σε περιβάλλοντα που υποστηρίζουν μόνο μορφές bitmap, διατηρώντας την αρχική εμφάνιση.

## Πώς να Μετατρέψετε HTML σε BMP με Aspose.HTML;
Για να μετατρέψετε HTML σε BMP με Aspose.HTML, πρώτα φορτώστε το αρχείο HTML σε ένα `HTMLDocument`. Στη συνέχεια, δημιουργήστε ένα αντικείμενο `ImageSaveOptions` που καθορίζει τη μορφή BMP και τυχόν επιθυμητές ρυθμίσεις ανάλυσης. Έπειτα, ορίστε τη διαδρομή εξόδου όπου θα γραφτεί το bitmap. Τέλος, καλέστε `Document.save` με τις επιλογές για να αποδώσετε και να αποθηκεύσετε την εικόνα. Αυτή η ροή εργασίας διαχειρίζεται αυτόματα CSS, γραμματοσειρές και SVG.

Παρακάτω υπάρχει ένας σύντομος, αριθμημένος οδηγός που σας καθοδηγεί σε κάθε ενέργεια. Τα μπλοκ κώδικα είναι ακριβώς τα ίδια με το αρχικό tutorial· προσθέσαμε μόνο περιεχόμενο και εξηγήσεις.

### Βήμα 1: Εισαγωγή Πακέτων Aspose.HTML για Java
`HTMLDocument` είναι η βασική κλάση που αντιπροσωπεύει ένα αρχείο HTML φορτωμένο στη μνήμη για απόδοση.  
```java
// Source HTML document
com.aspose.html.HTMLDocument htmlDocument = new com.aspose.html.HTMLDocument("path/to/your/input.html");
```

Δημιουργούμε μια παρουσία `HTMLDocument` που αντιπροσωπεύει το HTML που θέλετε να αποδώσετε. Αντικαταστήστε το `"path/to/your/input.html"` με την πραγματική διαδρομή του αρχείου.

### Βήμα 2: Αρχικοποίηση ImageSaveOptions για BMP
`ImageSaveOptions` ορίζει τη μορφή raster, την ανάλυση και τις ρυθμίσεις ποιότητας για την εικόνα εξόδου.  
```java
// Initialize ImageSaveOptions
com.aspose.html.saving.ImageSaveOptions options = new com.aspose.html.saving.ImageSaveOptions(com.aspose.html.rendering.image.ImageFormat.Bmp);
```

`ImageSaveOptions` ενημερώνει το Aspose.HTML ποια μορφή raster να παραγάγει. Εδώ καθορίζουμε `Bmp`, αλλά μπορείτε να το αλλάξετε σε PNG, JPEG κ.λπ., αν αργότερα χρειαστείτε διαφορετική μορφή **java html to image**.

### Βήμα 3: Ορισμός Διαδρομής Αρχείου Εξόδου
`OutputPath` είναι μια απλή συμβολοσειρά που λέει στη βιβλιοθήκη πού να γράψει το παραγόμενο αρχείο BMP.  
```java
// Output file path
String outputFile = "path/to/your/output/HTMLtoBMP_Output.bmp";
```

Ορίστε τον προορισμό όπου θα αποθηκευτεί το αρχείο BMP. Προσαρμόστε τη διαδρομή ανάλογα με τη δομή του έργου σας.

### Βήμα 4: Εκτέλεση της Μετατροπής
Η μέθοδος `Document.save` γράφει το αποδοθέν περιεχόμενο σε αρχείο χρησιμοποιώντας τις παρεχόμενες επιλογές αποθήκευσης.  
`Document.save` εκτελεί τη γραμμή απόδοσης, μετατρέποντας το έγγραφο HTML στην εικόνα BMP που καθορίσατε.  
```java
// Convert HTML to BMP
com.aspose.html.converters.Converter.convertHTML(htmlDocument, options, outputFile);
```
