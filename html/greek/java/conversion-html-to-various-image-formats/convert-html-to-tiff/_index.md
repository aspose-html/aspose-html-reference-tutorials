## Εισαγωγή Πακέτων

Στο έργο Java, εισάγετε τις βασικές κλάσεις του Aspose.HTML:

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
```

Αυτές οι εισαγωγές σας δίνουν πρόσβαση στη φόρτωση εγγράφου, στις επιλογές αποθήκευσης εικόνας και στη μηχανή μετατροπής.

## Πώς να φορτώσετε έγγραφο HTML java

Φορτώστε το αρχείο πηγής με μια γραμμή κώδικα. Η κλάση `HTMLDocument` αναλύει το markup, επιλύει το CSS και προετοιμάζει τη σελίδα για απόδοση. Αυτόματα εντοπίζει την κωδικοποίηση χαρακτήρων, ακολουθεί σχετικές URL και φορτώνει εξωτερικούς πόρους όπως εικόνες και γραμματοσειρές, εξασφαλίζοντας ότι η παραγόμενη έξοδος ταιριάζει με την προβολή σε πρόγραμμα περιήγησης. Αυτό καθιστά τη διαδικασία μετατροπής απλή και αξιόπιστη.

```java
HTMLDocument htmlDocument = new HTMLDocument("path/to/your/input.html");
```

Αντικαταστήστε το `"path/to/your/input.html"` με την πραγματική διαδρομή του αρχείου HTML σας. Το API ανιχνεύει αυτόματα την κωδικοποίηση χαρακτήρων και διαχειρίζεται εξωτερικούς πόρους όπως εικόνες ή γραμματοσειρές που αναφέρονται στο HTML.

## Πώς να μετατρέψετε HTML σε TIFF (render html tiff)

Ρυθμίστε τη μετατροπή ώστε να παραγάγει εικόνα TIFF. Η παράμετρος `ImageFormat` καθορίζει τον τύπο εξόδου, όπως TIFF, PNG ή JPEG, και σας επιτρέπει να επιλέξετε τη σωστή συμπίεση. Το TIFF επιλέγεται συχνά για τη lossless συμπίεση και την υποστήριξη πολυ‑σελίδων εγγράφων, καθιστώντας το ιδανικό για αρχειοθέτηση και εκτύπωση. Η ρύθμιση DPI και βάθους χρώματος μπορεί να βελτιώσει περαιτέρω την ποιότητα.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Tiff);
```

Αν αργότερα θέλετε **html to png java**, απλώς αλλάξτε το `ImageFormat.Tiff` σε `ImageFormat.Png`. Η μηχανή μετατροπής διατηρεί την αρχική ποιότητα vector, ώστε η παραγόμενη εικόνα να ταιριάζει με την απόδοση στην οθόνη.

## Πώς να ορίσετε τη διαδρομή εξόδου του αρχείου

Ορίστε πού θα αποθηκευτεί η παραγόμενη εικόνα. Η κλάση `File` αντιπροσωπεύει μια διαδρομή αρχείου ή φακέλου στο σύστημα αρχείων, επιτρέποντάς σας να καθορίσετε την ακριβή θέση εξόδου. Το αντικείμενο `File` δείχνει σε μια θέση στο σύστημα αρχείων σας, και η βιβλιοθήκη δημιουργεί αυτόματα τυχόν ελλείποντες φακέλους. Μπορείτε επίσης να ορίσετε δικαιώματα εγγραφής αν χρειαστεί.

```java
String outputFile = "path/to/your/output.tif";
```

Αλλάξτε την επέκταση αρχείου αν επιλέξετε διαφορετική μορφή· η ίδια μέθοδος `save` λειτουργεί για PNG, JPEG, BMP και GIF.

## Πώς να εκτελέσετε τη μετατροπή

Εκτελέστε τη μετατροπή με μία κλήση API. Η μέθοδος `save` γράφει την rasterized εικόνα στη διαδρομή προορισμού και απελευθερώνει όλους τους εγγενείς πόρους. Η κλάση `ImageSaveOptions` σας επιτρέπει να ορίσετε παραμέτρους εξόδου όπως μορφή, ανάλυση και συμπίεση. Παίρνει το αντικείμενο `File` προορισμού και το `ImageSaveOptions`, εφαρμόζοντας τη μορφή και τις ρυθμίσεις ποιότητας. Μετά την ολοκλήρωση, το έγγραφο κλείνει και η μνήμη ελευθερώνεται, εξασφαλίζοντας αποδοτική μαζική επεξεργασία.

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

Μετά την εκτέλεση, θα βρείτε το αρχείο TIFF στη θέση που καθορίσατε. Η λειτουργία ολοκληρώνεται σε λίγα χιλιοστά του δευτερολέπτου για τυπικά HTML μεγέθους ιστοσελίδας και κλιμακώνεται γραμμικά για μεγαλύτερα έγγραφα.
