## Εισαγωγή Πακέτων

Το πρώτο που πρέπει να κάνουμε είναι να φέρουμε τις απαιτούμενες κλάσεις του Aspose.HTML στο έργο μας. Αυτό το βήμα διασφαλίζει ότι ο μεταγλωττιστής γνωρίζει πού να βρει τα API μετατροπής.

### Βήμα 1: Εισαγωγή Πακέτων Aspose.HTML

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
```

Με τα πακέτα εισαχθέντα, είμαστε έτοιμοι να φορτώσουμε το HTML και να ξεκινήσουμε τη μετατροπή.

## Οδηγός Μετατροπής Βήμα‑βήμα

Παρακάτω διασπάμε όλη τη διαδικασία σε σαφή, αριθμημένα βήματα. Κάθε βήμα περιλαμβάνει μια σύντομη εξήγηση ακολουθούμενη από τον ακριβή κώδικα που πρέπει να αντιγράψετε.

### Βήμα 2: Φόρτωση του Πηγαίου HTML Εγγράφου

Δημιουργήστε ένα αντικείμενο `HTMLDocument` που δείχνει στο αρχείο εισόδου σας. Το Aspose.HTML διαβάζει το αρχείο, αναλύει το markup και δημιουργεί ένα DOM έτοιμο για απόδοση.

```java
HTMLDocument htmlDocument = new HTMLDocument("input.html");
```

> **Συμβουλή:** Αντικαταστήστε το `"input.html"` με την απόλυτη ή σχετική διαδρομή προς το πραγματικό σας αρχείο HTML.

### Βήμα 3: Ρύθμιση ImageSaveOptions για JPEG

Το `ImageSaveOptions` λέει στον μετατροπέα ποια μορφή εικόνας να παραγάγει και σας επιτρέπει να ρυθμίσετε παραμέτρους όπως η ποιότητα και η ανάλυση. Εδώ ορίζουμε το JPEG ως μορφή προορισμού.

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Jpeg);
```

> **Pro tip:** Αν χρειαστεί ποτέ να δημιουργήσετε PNG αντί για JPEG, απλώς αλλάξτε το `ImageFormat.Jpeg` σε `ImageFormat.Png`. Αυτό επίσης καλύπτει τη δευτερεύουσα λέξη‑κλειδί **java convert html png**.

### Βήμα 4: Ορισμός Διαδρομής Αρχείου Εξόδου

Αποφασίστε πού θα αποθηκευτεί το παραγόμενο JPEG. Μπορείτε να συμπεριλάβετε οποιονδήποτε φάκελο θέλετε· απλώς βεβαιωθείτε ότι η εφαρμογή έχει δικαιώματα εγγραφής.

```java
String outputFile = "HTMLtoJPEG_Output.jpeg";
```

Αλλάξτε το όνομα του αρχείου ή την επέκταση εάν στοχεύετε σε διαφορετικό τύπο εικόνας.

### Βήμα 5: Εκτέλεση της Μετατροπής

Τέλος, καλέστε τη στατική μέθοδο `convertHTML`. Παίρνει το φορτωμένο έγγραφο, τις επιλογές αποθήκευσης και τη διαδρομή προορισμού, και γράφει το JPEG στο δίσκο.

```java
Converter.convertHTML(htmlDocument, options, outputFile);
```

Αυτό είναι! Όταν το πρόγραμμα ολοκληρωθεί, θα βρείτε μια απόδοση JPEG της σελίδας HTML στη θέση που ορίσατε.
