## Εισαγωγή Πακέτων

Για να εργαστείτε με την Aspose.HTML, εισάγετε τις απαιτούμενες κλάσεις:

```java
import com.aspose.html.HTMLDocument;

```

## Οδηγός Βήμα‑βήμα για τη Μετατροπή HTML σε PNG

Παρακάτω υπάρχει ένας σαφής, αριθμημένος οδηγός που δείχνει ακριβώς πώς να **generate png from html** χρησιμοποιώντας την Aspose.HTML.

### Βήμα 1: Φόρτωση του HTML Εγγράφου

Αρχικά, δημιουργήστε ένα αντικείμενο `HTMLDocument` που δείχνει στο πηγαίο αρχείο σας.

```java
// Source HTML document
HTMLDocument htmlDocument = new HTMLDocument("input.html");
```

### Βήμα 2: Διαμόρφωση ImageSaveOptions

Ορίστε το `ImageSaveOptions` ώστε να καθορίζει το PNG ως μορφή εξόδου.

```java
// Initialize ImageSaveOptions
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Png);
```

Μπορείτε επίσης να προσαρμόσετε το `options` (π.χ., πλάτος, ύψος, ποιότητα) εάν χρειάζεστε προσαρμοσμένες διαστάσεις.

### Βήμα 3: Ορισμός Διαδρομής Εξόδου

Επιλέξτε πού θα αποθηκευτεί η αποδοθείσα εικόνα.

```java
// Output file path
String outputFile = "HTMLtoPNG_Output.png";
```

Αλλάξτε το όνομα αρχείου ή τον φάκελο ώστε να ταιριάζει στη δομή του έργου σας.

### Βήμα 4: Εκτέλεση της Μετατροπής

Τέλος, καλέστε τον μετατροπέα για να αποδώσετε και να αποθηκεύσετε το PNG.

```java
// Convert HTML to PNG
Converter.convertHTML(htmlDocument, options, outputFile);
```

Όταν εκτελεστεί αυτή η γραμμή, η Aspose.HTML επεξεργάζεται το HTML, εφαρμόζει το CSS, επιλύει τους πόρους και γράφει ένα υψηλής ποιότητας αρχείο PNG στο `outputFile`.
