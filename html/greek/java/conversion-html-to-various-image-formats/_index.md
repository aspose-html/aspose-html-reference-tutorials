## Πώς να **convert html gif**, **convert html jpeg**, και **convert html png** με το Aspose.HTML

Παρακάτω θα βρείτε σύντομες οδηγίες βήμα‑βήμα για κάθε μορφή εξόδου. Τα αποσπάσματα κώδικα είναι τα ίδια εκτός από την επέκταση του αρχείου εξόδου· αρκεί να αλλάξετε τη μορφή στο `ImageSaveOptions`.

### Βήμα 1: Ρύθμιση του HTML renderer
Δημιουργήστε ένα αντικείμενο `HtmlRenderer` και φορτώστε την πηγή HTML (αρχείο, URL ή συμβολοσειρά).

### Βήμα 2: Επιλογή της επιθυμητής μορφής εικόνας
Επιλέξτε μία από τις υποκλάσεις του `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions`, κ.λπ.

### Βήμα 3: Απόδοση και αποθήκευση
Καλείτε `renderer.renderToFile(outputPath, saveOptions)` για να δημιουργήσετε την εικόνα.

*Τα ίδια τρία βήματα ισχύουν και για BMP, TIFF και οποιαδήποτε άλλη υποστηριζόμενη μορφή.*
