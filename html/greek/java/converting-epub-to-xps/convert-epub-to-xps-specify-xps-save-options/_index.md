# Καθορισμός Επιλογών Αποθήκευσης Aspose HTML για EPUB σε XPS

Η μετατροπή ενός εγγράφου EPUB σε XPS μπορεί να είναι ένα χρήσιμο μέρος μιας διαδικασίας δημοσίευσης, και οι **Aspose HTML Save Options** σας παρέχουν λεπτομερή έλεγχο του αποτελέσματος. Σε αυτό το tutorial θα περάσουμε βήμα‑βήμα τις ακριβείς ενέργειες που απαιτούνται για τη διαμόρφωση αυτών των επιλογών αποθήκευσης με το Aspose.HTML for Java και τη δημιουργία ενός υψηλής ποιότητας αρχείου XPS.

## Γρήγορες Απαντήσεις
- **Τι κάνει οι Aspose HTML Save Options;** Σας επιτρέπει να προσαρμόσετε το μέγεθος σελίδας, το χρώμα φόντου και άλλες ρυθμίσεις απόδοσης κατά την αποθήκευση σε μορφές όπως το XPS.  
- **Ποια βιβλιοθήκη απαιτείται;** Aspose.HTML for Java (τελευταία έκδοση).  
- **Χρειάζομαι άδεια;** Μια δωρεάν δοκιμή λειτουργεί για αξιολόγηση· απαιτείται εμπορική άδεια για παραγωγή.  
- **Μπορώ να αλλάξω τις διαστάσεις της σελίδας;** Ναι – μπορείτε να ορίσετε οποιοδήποτε πλάτος και ύψος μέσω του `PageSetup`.  
- **Είναι η μετατροπή γρήγορη;** Για τυπικά αρχεία EPUB η μετατροπή ολοκληρώνεται σε δευτερόλεπτα σε μια σύγχρονη JVM.

## Τι είναι οι Aspose HTML Save Options;
Οι Aspose HTML Save Options είναι ένα σύνολο ιδιοτήτων που ελέγχουν πώς τα HTML, EPUB ή άλλα έγγραφα βασισμένα στο web αποδίδονται και αποθηκεύονται σε μορφή-στόχο όπως XPS, PDF ή αρχεία εικόνας. Σας επιτρέπουν να ορίσετε τη διάταξη της σελίδας, τα χρώματα φόντου, τα περιθώρια και άλλα, εξασφαλίζοντας ότι το παραγόμενο αρχείο ταιριάζει με τις απαιτήσεις του σχεδίου σας.

- **Προγραμματιστικός έλεγχος** – αυτοματοποιήστε μαζικές μετατροπές μέσα σε εφαρμογές Java.

- **Προσαρμόσιμο αποτέλεσμα** – χάρη στις επιλογές αποθήκευσης μπορείτε να προσαρμόσετε το XPS ακριβώς στις ανάγκες σας.

Έχοντας αυτά έτοιμα θα μπορείτε να ακολουθήσετε τα βήματα χωρίς διακοπές.

## Εισαγωγή Πακέτων

Πρώτα, εισάγετε τις απαιτούμενες κλάσεις. Τοποθετήστε τις εισαγωγές στην αρχή του αρχείου πηγαίου κώδικα Java:

```java
import java.io.FileInputStream;
import com.aspose.html.saving.XpsSaveOptions;
import com.aspose.html.rendering.PageSetup;
import com.aspose.html.drawing.Page;
import com.aspose.html.drawing.Size;
import com.aspose.html.drawing.Length;
import com.aspose.html.drawing.Color;
import com.aspose.html.converters.Converter;
import com.aspose.html.system.io.resources.Resources;
```

Αυτές οι εισαγωγές σας δίνουν πρόσβαση στη μηχανή μετατροπής και στις **Aspose HTML Save Options** που απαιτούνται για λεπτομερή ρύθμιση.

## Άνοιγμα του Αρχείου EPUB

Στη συνέχεια, ανοίξτε το EPUB που θέλετε να μετατρέψετε. Η βοηθητική λειτουργία `Resources.input` φορτώνει ένα αρχείο από το φάκελο πόρων της επίδειξης:

```java
try (FileInputStream fileInputStream = new FileInputStream(Resources.input("input.epub"))) {
    // Your code for the EPUB file handling goes here.
}
```

Η χρήση ενός μπλοκ try‑with‑resources εξασφαλίζει ότι η ροή κλείνει αυτόματα.

## Δημιουργία XPS Save Options (Διαμόρφωση Μεγέθους Σελίδας & Φόντου)

Τώρα δημιουργούμε ένα αντικείμενο `XpsSaveOptions` και το προσαρμόζουμε. Εδώ οι **save options** πραγματικά ξεχωρίζουν:

```java
XpsSaveOptions options = new XpsSaveOptions();
PageSetup pageSetup = new PageSetup();
Page anyPage = new Page();
anyPage.setSize(new Size(Length.fromPixels(3000), Length.fromPixels(1000)));
pageSetup.setAnyPage(anyPage);
options.setPageSetup(pageSetup);
options.setBackgroundColor(Color.getAliceBlue());
```

- **Μέγεθος σελίδας** – ορίζεται σε 3000 × 1000 pixel σε αυτό το παράδειγμα· προσαρμόστε το ώστε να ταιριάζει με το επιθυμητό μέγεθος χαρτιού.  
- **Χρώμα φόντου** – το `AliceBlue` δείχνει πώς να αλλάξετε το φόντο του καμβά· μπορείτε να επιλέξετε οποιοδήποτε `Color`.  

Αυτές οι ρυθμίσεις αποτελούν μέρος των **Aspose HTML Save Options** που επηρεάζουν το τελικό έγγραφο XPS.

## Μετατροπή EPUB σε XPS

Τέλος, καλέστε τον μετατροπέα με τη ροή, τις διαμορφωμένες επιλογές και μια διαδρομή εξόδου:

```java
Converter.convertEPUB(
    fileInputStream,
    options,
    Resources.output("output.xps")
);
```

Όταν εκτελεστεί αυτή η γραμμή, το Aspose.HTML διαβάζει το EPUB, εφαρμόζει τη ρύθμιση σελίδας και το χρώμα φόντου που ορίσατε, και γράφει ένα πλήρως συμβατό αρχείο XPS.

## Συχνά Προβλήματα & Συμβουλές

- **Λανθασμένες διαστάσεις σελίδας** – βεβαιωθείτε ότι οι διαστάσεις εκφράζονται σε pixel (ή χρησιμοποιήστε `Length.fromInches`).  
- **Λείπουν γραμματοσειρές** – ενσωματώστε τις απαιτούμενες γραμματοσειρές στο EPUB ή εγκαταστήστε τις στον κεντρικό υπολογιστή JVM για να αποφύγετε την εναλλακτική.  
- **Μεγάλα αρχεία** – για πολύ μεγάλα EPUB, αυξήστε τη μνήμη heap της JVM (`-Xmx`) για να αποτρέψετε `OutOfMemoryError`.
