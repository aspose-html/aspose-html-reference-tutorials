## Συχνές Ερωτήσεις

**Q: Πώς μετατρέπω EPUB σε JPG χρησιμοποιώντας το Aspose.HTML;**  
A: Φορτώστε το EPUB με `HtmlDocument`, ορίστε τη μορφή `ImageSaveOptions` σε `Jpeg` και καλέστε `save`.

**Q: Μπορώ να μετατρέψω EPUB σε PNG χωρίς απώλεια ποιότητας;**  
A: Ναι, το PNG είναι χωρίς απώλειες· απλώς επιλέξτε `Png` ως μορφή εξόδου στο `ImageSaveOptions`.

**Q: Ποιος είναι ο καλύτερος τρόπος για να μετατρέψετε EPUB σε GIF για κινούμενες εικόνες;**  
A: Αποδώστε κάθε σελίδα ως ξεχωριστό καρέ GIF ή συνδυάστε τα χρησιμοποιώντας έναν εξωτερικό κωδικοποιητή GIF μετά τη μετατροπή.

**Q: Πώς μπορώ να προσαρμόσω τη ροή εξόδου για μεγάλα αρχεία EPUB;**  
A: Υλοποιήστε έναν προσαρμοσμένο `IStreamProvider` και περάστε τον στη μέθοδο `save` για να κατευθύνετε την έξοδο σε μνήμη, cloud ή άλλη αποθήκευση.

**Q: Είναι πραγματικά δυνατή η μετατροπή με μία γραμμή;**  
A: Απόλυτα – `new HtmlDocument("input.epub").save("output.bmp", new ImageSaveOptions(SaveFormat.Bmp));`
