## Εισαγωγή Πακέτων

Στον κώδικά σας Java, πρέπει να εισάγετε τα απαραίτητα πακέτα για να χρησιμοποιήσετε το Aspose.HTML για Java αποτελεσματικά. Δείτε πώς:

```java
import com.aspose.html.saving.ImageSaveOptions;
import com.aspose.html.rendering.image.ImageFormat;
import com.aspose.html.converters.Converter;
```

## Μετατροπή EPUB σε BMP – Βήμα‑βήμα Μετατροπή

### Βήμα 1: Άνοιγμα του Αρχείου EPUB

Πρώτα, πρέπει να ανοίξετε ένα υπάρχον αρχείο EPUB για ανάγνωση. Μπορείτε να το κάνετε χρησιμοποιώντας τον παρακάτω κώδικα:

```java
try (java.io.FileInputStream fileInputStream = new java.io.FileInputStream("path/to/your/input.epub")) {
```

Αντικαταστήστε το `"path/to/your/input.epub"` με το πραγματικό μονοπάτι του αρχείου EPUB σας.

### Βήμα 2: Αρχικοποίηση ImageSaveOptions

Στη συνέχεια, αρχικοποιήστε το `ImageSaveOptions` με τη ζητούμενη μορφή εικόνας (σε αυτήν την περίπτωση, BMP):

```java
ImageSaveOptions options = new ImageSaveOptions(ImageFormat.Bmp);
```

Αυτό το βήμα εξασφαλίζει ότι η έξοδος θα είναι σε μορφή BMP.

#### Προσαρμογή DPI BMP (Δευτερεύουσα Λέξη-Κλειδί)

Αν χρειάζεστε εικόνες υψηλότερης ανάλυσης, μπορείτε να **προσαρμόσετε το DPI του BMP** ορίζοντας την ανάλυση στο ίδιο αντικείμενο `options`, π.χ., `options.setResolution(300);`. Ρυθμίστε την τιμή DPI ώστε να καλύπτει τις απαιτήσεις ποιότητας πριν καλέσετε τη μέθοδο μετατροπής.

### Βήμα 3: Μετατροπή EPUB σε BMP

Τώρα, καλέστε τη μέθοδο `convertEPUB` για να μετατρέψετε το αρχείο EPUB σε BMP. Αυτό το απόσπασμα κώδικα θα διαχειριστεί τη διαδικασία μετατροπής:

```java
Converter.convertEPUB(fileInputStream, options, "path/to/your/output.bmp");
```

Αντικαταστήστε το `"path/to/your/output.bmp"` με το πραγματικό μονοπάτι όπου θέλετε να αποθηκεύσετε το αρχείο BMP.

Και αυτό είναι—έχετε μετατρέψει επιτυχώς **epub σε bmp** χρησιμοποιώντας Aspose.HTML για Java.
