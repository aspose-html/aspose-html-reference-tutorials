# Δημιουργία HTML από Κώδικα με Aspose.HTML για Java και Αποθήκευση

Είτε δημιουργείτε δυναμικές αναφορές, πρότυπα email ή ιστοσελίδες σε πραγματικό χρόνο, τα παρακάτω βήματα θα σας θέσουν σε λειτουργία μέσα σε λίγα λεπτά.

## Οδηγός Βήμα-Βήμα

### Βήμα 1: Προετοιμάστε τον HTML Κώδικά σας
Πρώτα, γράψτε το HTML markup που θέλετε να μετατρέψετε σε έγγραφο. Αυτό μπορεί να είναι οποιοδήποτε έγκυρο απόσπασμα HTML.

```java
String html_code = "<p>Hello World!</p>";
```

Εδώ αποθηκεύουμε μια απλή παράγραφο στη μεταβλητή `html_code`. Σκεφτείτε το ως το σχέδιο για τη σελίδα που πρόκειται να δημιουργήσετε.

### Βήμα 2: Αρχικοποιήστε το Έγγραφο από τη Μεταβλητή Συμβολοσειράς
Στη συνέχεια, δημιουργήστε ένα αντικείμενο `HTMLDocument` χρησιμοποιώντας τη συμβολοσειρά. Εδώ είναι που **convert string to HTML**.

```java
com.aspose.html.HTMLDocument document = new com.aspose.html.HTMLDocument(html_code, ".");
```

Το `"."` λέει στο Aspose να χρησιμοποιήσει τον τρέχοντα φάκελο ως βασική διαδρομή. Τώρα έχετε ένα HTML έγγραφο στη μνήμη που μπορείτε να επεξεργαστείτε περαιτέρω αν χρειαστεί.

### Βήμα 3: Αποθηκεύστε το Έγγραφο στο Δίσκο
Τέλος, γράψτε το έγγραφο σε ένα φυσικό αρχείο. Αυτό είναι το βήμα **save html file java**.

```java
document.save("create-from-string.html");
```

Το αρχείο `create-from-string.html` θα εμφανιστεί στον ίδιο φάκελο όπου εκτελείτε το πρόγραμμα. Μπορείτε να το ανοίξετε σε οποιονδήποτε περιηγητή για να δείτε το αποτέλεσμα.
