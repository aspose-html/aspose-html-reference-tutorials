## Masalah Umum dan Solusi
- **Font tidak muncul di PDF** – Pastikan font yang diperlukan terpasang di server atau sematkan menggunakan `PdfSaveOptions`.  
- **File EPUB besar menyebabkan tekanan memori** – Gunakan pemrosesan berbasis aliran (`MemoryStream` → `FileStream`) untuk mengurangi penggunaan heap.  
- **Rendering kanvas muncul kosong** – Pastikan kanvas telah sepenuhnya digambar sebelum memanggil API konversi.
