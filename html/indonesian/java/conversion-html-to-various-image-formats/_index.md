## Cara **convert html gif**, **convert html jpeg**, dan **convert html png** dengan Aspose.HTML

Berikut ini Anda akan menemukan instruksi singkat langkah‑demi‑langkah untuk setiap format target. Potongan kode identik kecuali ekstensi file output; Anda hanya perlu mengubah format pada `ImageSaveOptions`.

### Langkah 1: Siapkan renderer HTML
Buat instance `HtmlRenderer` dan muat sumber HTML (file, URL, atau string).

### Langkah 2: Pilih format gambar yang diinginkan
Pilih salah satu subclass `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions`, dll.

### Langkah 3: Render dan simpan
Panggil `renderer.renderToFile(outputPath, saveOptions)` untuk menghasilkan gambar.

*Ketiga langkah yang sama berlaku untuk BMP, TIFF, dan format lain yang didukung.*

## Tips Pemecahan Masalah
- **Output kosong** – Pastikan sumber HTML telah dimuat sepenuhnya; gunakan `renderer.waitForComplete()` bila diperlukan.

- **Font tidak ditemukan** – Instal font yang diperlukan di server atau sematkan dalam HTML menggunakan `@font-face`.
