## Как **convert html gif**, **convert html jpeg** и **convert html png** с помощью Aspose.HTML

Ниже представлены краткие пошаговые инструкции для каждого целевого формата. Фрагменты кода идентичны, за исключением расширения выходного файла; необходимо лишь изменить формат в `ImageSaveOptions`.

### Шаг 1: Настройка HTML‑рендерера
Создайте экземпляр `HtmlRenderer` и загрузите HTML‑источник (файл, URL или строку).

### Шаг 2: Выбор желаемого формата изображения
Выберите один из подклассов `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions` и т.д.

### Шаг 3: Рендеринг и сохранение
Вызовите `renderer.renderToFile(outputPath, saveOptions)`, чтобы сгенерировать изображение.

*Те же три шага применимы к BMP, TIFF и любому другому поддерживаемому формату.*

- **Пустой вывод** – убедитесь, что HTML‑источник полностью загружен; при необходимости используйте `renderer.waitForComplete()`.

- **Отсутствие шрифтов** – установите необходимые шрифты на сервере или внедрите их в HTML с помощью `@font-face`.
