## Как генерировать PDF из SVG с помощью Aspose.HTML for Java

### Импорт пакетов
Сначала импортируйте необходимые классы в ваш Java‑проект.

```java
import com.aspose.html.dom.svg.SVGDocument;
import com.aspose.html.saving.PdfSaveOptions;
import com.aspose.html.converters.Converter;
```

### Шаг 1: Загрузка SVG‑документа
Чтобы начать конвертацию, загрузите исходный SVG‑файл в объект `SVGDocument`. Это демонстрирует **как загрузить SVG‑документ в стиле Java**.

```java
SVGDocument svgDocument = new SVGDocument(Resources.input("input.svg"));
```

### Шаг 2: Настройка параметров сохранения PDF
Настройте параметры вывода PDF. Например, вы можете задать качество JPEG, чтобы обеспечить наилучший визуальный результат при растеризации изображений внутри SVG.

```java
PdfSaveOptions options = new PdfSaveOptions();
options.setJpegQuality(100);
```

### Шаг 3: Определение пути вывода
Укажите, где должен быть сохранён сгенерированный PDF. Убедитесь, что каталог существует и у вас есть права на запись.

```java
String outputFile = Resources.output("SVGtoPDF_Output.pdf");
```

### Шаг 4: Конвертация SVG в PDF
Наконец, выполните конвертацию, вызвав метод `convertSVG`. Этот шаг **подготавливает векторную графику к PDF**.

```java
Converter.convertSVG(svgDocument, options, outputFile);
```

Поздравляем! Вы успешно **сгенерировали PDF из SVG** с помощью Aspose.HTML for Java. Полученный PDF можно найти по пути, указанному в `outputFile`.

## Распространённые проблемы и решения
- **Отсутствуют шрифты или стили:** Убедитесь, что все внешние шрифты, указанные в SVG, установлены в системе или встроены в SVG.  
- **Ошибки доступа:** Проверьте, что ваш процесс Java имеет права записи в каталог вывода.  
- **Большие SVG‑файлы:** При возникновении `OutOfMemoryError` рассмотрите возможность увеличения размера кучи JVM (`-Xmx`).
