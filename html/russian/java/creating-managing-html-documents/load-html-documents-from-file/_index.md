## Что такое “how to load html” в Java?
Загрузка HTML‑файла означает создание объекта `HTMLDocument`, представляющего DOM файла. После загрузки вы можете читать элементы, изменять содержимое или конвертировать документ в другие форматы, такие как PDF или изображение.

## Почему использовать Aspose.HTML для Java?
Aspose.HTML предоставляет высокопроизводительный кроссплатформенный API, который работает с современными HTML5, CSS3 и JavaScript. Он скрывает детали низкоуровневого парсинга, позволяя сосредоточиться на бизнес‑логике, а не на нюансах разбора HTML.

## Создание HTML‑файла
Прежде чем мы сможем загрузить HTML‑файл, его нужно создать. Считайте это подготовкой холста перед началом вашего шедевра.

### Шаг 1: Создать HTML‑файл
В теле вашей Java‑программы создадим быстрый HTML‑файл с именем **load‑from‑file.html** со следующим кодом:
```java
try (FileWriter fileWriter = new FileWriter("load-from-file.html")) {
    fileWriter.write("<html><body><h1>Hello World!</h1></body></html>");
}
```
Этот фрагмент:
- Открывает (или создаёт) файл с именем `load-from-file.html`.
- Записывает минимальную HTML‑структуру, содержащую заголовок “Hello World!”.

### Шаг 2: Загрузить HTML‑файл
После создания HTML‑файла следующий шаг — загрузить его в программу:
```java
com.aspose.html.HTMLDocument document = new com.aspose.html.HTMLDocument("load-from-file.html");
```
Инициализируя объект `HTMLDocument` с путем к файлу, вы позволяете библиотеке Aspose.HTML прочитать HTML‑содержимое.

### Шаг 3: Вывести загруженный документ
Чтобы убедиться, что всё работает корректно, выведем содержимое документа в консоль:
```java
System.out.println(document.getDocumentElement().getOuterHTML());
```

## Часто задаваемые вопросы
**Q: Можно ли загрузить HTML‑строку вместо файла?**  
A: Да, вы можете создать `HTMLDocument` из `String` или `InputStream`, содержащего HTML‑разметку.

**Q: Поддерживает ли Aspose.HTML выполнение CSS и JavaScript?**  
A: Он полностью парсит CSS для расчётов макета, но не выполняет JavaScript. При необходимости выполнения скриптов используйте безголовый браузер.

**Q: Как конвертировать загруженный HTML в PDF?**  
A: После загрузки используйте `com.aspose.html.rendering.pdf.PdfSaveOptions` вместе с методом `document.save()` для создания PDF‑файла.

**Q: Можно ли изменить DOM после загрузки?**  
A: Конечно. Вы можете использовать стандартные методы DOM, такие как `getElementById`, `createElement` и `appendChild`, на объекте `HTMLDocument`.

**Q: Какие требования к памяти существуют для больших HTML‑файлов?**  
A: Для очень больших файлов рекомендуется загружать документ в потоковом режиме или увеличить размер кучи JVM.
