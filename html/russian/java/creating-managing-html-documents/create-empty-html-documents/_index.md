## Как создать HTML‑документы – пошаговое руководство

### Шаг 1: Инициализация HTML‑документа
Сначала создайте экземпляр пустого документа. Это даст вам чистый холст для работы.

```java
com.aspose.html.HTMLDocument document = new com.aspose.html.HTMLDocument();
```

Строка выше создаёт новый `HTMLDocument`. На данный момент документ не содержит разметки, что идеально, когда вы хотите построить структуру с нуля.

### Шаг 2: Сохранение документа на диск
После инициализации документа вы захотите сохранить его как файл `.html`. Используйте метод `save` и обязательно очистите ресурсы, чтобы **prevent memory leaks java**.

```java
try {
    document.save("create-empty-document.html");
} finally {
    if (document != null) {
        document.dispose();
    }
}
```

- `document.save("create-empty-document.html")` записывает пустой HTML‑файл по указанному пути.  
- Блок `finally` гарантирует, что `document.dispose()` выполнится даже при возникновении исключения, помогая избежать утечек памяти.

### Распространённые ошибки и советы
- **Никогда не пропускайте вызов `dispose()`** – забывание этого шага может привести к утечкам памяти, особенно в длительно работающих приложениях.  
- **Путь к файлу имеет значение** – указывайте абсолютный путь или убедитесь, что текущий каталог доступен для записи, чтобы избежать `IOException`.  
- **Переиспользуйте тот же шаблон** – тот же шаблон инициализации и освобождения работает для более сложных сценариев, таких как добавление узлов DOM или конвертация в PDF.
