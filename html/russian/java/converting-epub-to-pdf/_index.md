## Часто задаваемые вопросы

**В:** *Могу ли я конвертировать защищённые файлы EPUB в PDF?*  
**О:** Да. Aspose.HTML for Java может открывать зашифрованные EPUB, если вы предоставите соответствующий пароль расшифровки.

**В:** *Что если мне нужен PDF, соответствующий PDF/A?*  
**О:** Используйте класс `PdfSaveOptions`, чтобы установить `Compliance` в `PdfCompliance.PdfA1b` (или другие уровни PDF/A) перед сохранением.

**В:** *Можно ли извлекать изображения, такие как JPG или PNG, во время конвертации?*  
**О:** Абсолютно. Указав параметры сохранения изображения или используя пользовательский провайдер потоков, вы можете направлять каждую страницу или изображение в формат JPG или PNG.

**В:** *Нужна ли отдельная библиотека для извлечения изображений?*  
**О:** Нет. Один и тот же API Aspose.HTML for Java обрабатывает как генерацию PDF, так и извлечение изображений, упрощая рабочий процесс.

**В:** *Какие версии Java поддерживаются?*  
**О:** Библиотека поддерживает Java 8 и новее, включая Java 11, Java 17 и последующие LTS‑версии.
