## Импорт пакетов

Сначала откройте ваш Java‑проект и импортируйте необходимые пакеты Aspose.HTML for Java. Эти импорты предоставляют доступ к конвертеру и параметрам, специфичным для PDF.

```java
import com.aspose.html.saving.PdfSaveOptions;
import com.aspose.html.converters.Converter;
import java.io.FileInputStream;
```

## Загрузка EPUB‑файла

Далее загрузите EPUB‑файл, который хотите конвертировать. Замените `"input.epub"` на реальный путь к вашему исходному файлу.

```java
try (FileInputStream fileInputStream = new FileInputStream("input.epub")) {
```

Оператор `try‑with‑resources` гарантирует автоматическое закрытие потока после конвертации.

## Установка параметров конвертации PDF

Создайте экземпляр `PdfSaveOptions` для тонкой настройки вывода PDF. Здесь вы можете регулировать размер страницы, поля и другие параметры рендеринга.

```java
PdfSaveOptions options = new PdfSaveOptions();
```

## Конвертация EPUB в PDF

Теперь выполните фактическую конвертацию. Метод `Converter.convertEPUB` читает поток EPUB, применяет параметры и записывает результат в `output.pdf`.

```java
Converter.convertEPUB(fileInputStream, options, "output.pdf");
```

После выполнения этой строки вы найдете `output.pdf` в каталоге вашего проекта, готовый к распространению или архивированию.

## Распространённые проблемы и решения

| Проблема | Причина | Решение |
|----------|---------|---------|
| **Пустые страницы в PDF** | Отсутствие CSS или шрифтов в EPUB | Убедитесь, что EPUB содержит все ресурсы, или внедрите шрифты через `PdfSaveOptions` |
| **Неправильный размер страницы** | Параметры по умолчанию не соответствуют вашим требованиям | Установите `options.setPageSize()` или `options.setMargins()` перед конвертацией |
| **OutOfMemoryError** при больших EPUB | Большие файлы потребляют слишком много памяти | Обрабатывайте EPUB частями или увеличьте размер кучи JVM (`-Xmx2g`) |
