## Как выполнить конвертацию epub to pdf java

Ниже представлено пошаговое руководство. Каждый шаг включает краткое объяснение, за которым следует оригинальный блок кода (без изменений).

### Шаг 1: Открыть исходный документ EPUB

Мы начинаем с открытия исходного файла EPUB с помощью `FileInputStream`. Этот поток передаёт конвертеру необработанные данные EPUB.

```java
try (java.io.FileInputStream fileInputStream = new java.io.FileInputStream("input.epub")) {
```

### Шаг 2: Инициализировать `PdfSaveOptions`

Создайте экземпляр `PdfSaveOptions` для настройки конвертации в PDF. Например, вы можете установить качество JPEG, чтобы изображения сохраняли высокую точность.

```java
PdfSaveOptions options = new PdfSaveOptions();
options.setJpegQuality(100);
```

### Шаг 3: Определить путь к выходному файлу

Укажите, где следует сохранить полученный PDF. При необходимости можно изменить имя файла или каталог.

```java
String outputFile = "EPUBtoPDF_Output.pdf";
```

### Шаг 4: Конвертировать EPUB в PDF

Наконец, вызовите метод `convertEPUB`, передав входной поток, параметры сохранения и путь к выходному файлу.

```java
Converter.convertEPUB(fileInputStream, options, outputFile);
```

## Распространённые ошибки и советы
- **Пути к файлам** — Используйте абсолютные пути или убедитесь, что рабочий каталог установлен правильно, чтобы избежать `FileNotFoundException`.
- **Использование памяти** — Для очень больших EPUB рассмотрите возможность потоковой записи вывода или увеличения размера кучи JVM.
- **Настройка вывода** — Изучите другие свойства `PdfSaveOptions`, такие как `setEmbedStandardFonts(true)` для встраивания шрифтов или `setPageSize(...)` для управления размерами страниц.
