# Как создать HTML со стилизованным текстом – Полное руководство по C#

Когда‑то задавались вопросом **как создать HTML** из C# без мучений с конкатенацией строк? Вы не одиноки. Во многих проектах нужно сгенерировать небольшой фрагмент разметки — может быть, тело письма или динамический отчёт — при этом сохранить стили чистыми и поддерживаемыми. Хорошая новость? С Aspose.HTML вы можете делать это программно, и вы увидите точно **как применить настройки шрифта**, **оформить текст полужирным курсивом**, и **добавить элемент абзаца** не выходя из IDE.

В этом руководстве мы пройдём каждый шаг, от инициализации пустого документа до сохранения финального файла. К концу вы сможете **генерировать стилизованный HTML**, который можно вставить в любую веб‑страницу, почтовый клиент или тело API‑запроса. Без внешних шаблонов, без грязных строковых трюков — только чистый C#‑код, понятный каждому.

## Что вы узнаете

- **Как создать HTML**‑документ с помощью Aspose.HTML.  
- Правильный способ **применить свойства шрифта** (размер, семейство, полужирный, курсив) с использованием `WebFontStyle`.  
- Как **добавить элемент абзаца** в DOM и задать его внутреннее содержимое.  
- Приёмы **оформления текста полужирным курсивом** в одну строку кода.  
- Как **генерировать стилизованный HTML** и проверить результат.

Предпосылки минимальны: .NET 6+ (или .NET Framework 4.6.2+), ссылка на библиотеку Aspose.HTML for .NET и любой удобный IDE. Если они у вас уже есть, приступим.

## Шаг 1 – Настройка проекта и импорт пространств имён

Прежде чем **создавать HTML**, нужен C#‑проект, ссылающийся на Aspose.HTML. Откройте Visual Studio (или ваш любимый редактор), создайте новое консольное приложение и добавьте пакет NuGet:

```bash
dotnet add package Aspose.HTML
```

Затем подключите необходимые пространства имён:

```csharp
using Aspose.Html;
using Aspose.Html.Drawing;
```

> **Совет:** Держите инструкции `using` в начале файла; так остальной код будет чище и легче восприниматься.

---

## Шаг 2 – Инициализация пустого HTML‑документа

Теперь, когда среда готова, можно создать новый `HTMLDocument`. Представьте его как чистый холст, на котором мы будем рисовать наш стилизованный абзац.

```csharp
// Step 2: Create a new empty HTML document
HTMLDocument doc = new HTMLDocument();
```

Почему начинать с пустого документа, а не загружать шаблон? Потому что это гарантирует полный контроль над каждым элементом и устраняет скрытые стили, которые могут помешать достижению цели **оформить текст полужирным курсивом**.

---

## Шаг 3 – Определение шрифта с полужирным и курсивным стилями

Aspose.HTML использует класс `Font` совместно с флагами `WebFontStyle` для описания внешнего вида текста. Вот строка, которая делает всю тяжёлую работу:

```csharp
// Step 3: Define a font with both bold and italic styles
Font font = new Font("Times New Roman", 14, WebFontStyle.Bold | WebFontStyle.Italic);
```

Оператор `|` объединяет два флага стиля, поэтому вы получаете один объект `Font`, который одновременно **полужирный и курсивный**. При желании можно добавить `WebFontStyle.Underline` для дополнительного эффекта.

---

## Шаг 4 – Создание элемента абзаца и применение шрифта

С готовым шрифтом создаём элемент `<p>`, привязываем шрифт к его стилю и заполняем его нашим сообщением.

```csharp
// Step 4: Create a paragraph element and apply the font to its style
HtmlElement paragraph = doc.CreateElement("p");
paragraph.Style.Font = font;

// Step 5: Set the paragraph's text content
paragraph.InnerHtml = "Bold‑italic text rendered with WebFontStyle.";
```

Обратите внимание, что свойство `Style.Font` принимает объект `Font` напрямую — никаких CSS‑строк не требуется. Такой подход типобезопасен и удобен для IDE, уменьшая шанс опечаток, часто встречающихся при ручном формировании HTML‑строк.

---

## Шаг 5 – Добавление абзаца в тело документа

Иерархия DOM имеет значение. Добавив абзац в `doc.Body`, вы гарантируете, что элемент появится в финальной разметке, как если бы вы редактировали HTML‑файл вручную.

```csharp
// Step 6: Add the paragraph to the document body
doc.Body.AppendChild(paragraph);
```

Если понадобится добавить другие элементы (таблицы, изображения, скрипты), повторяйте эту схему — создаёте, стилизуете, затем добавляете.

---

## Шаг 6 – Сохранение полученного HTML‑файла

Последний шаг — записать документ на диск. Выберите папку, в которую у вас есть права записи, и вызовите `Save`. На выходе получится чистый HTML‑файл, открываемый в любом браузере.

```csharp
// Step 7: Save the resulting HTML to view the styled text
string outputPath = Path.Combine(Environment.CurrentDirectory, "output.html");
doc.Save(outputPath);
Console.WriteLine($"HTML saved to: {outputPath}");
```

Открыв `output.html`, вы увидите предложение, отрисованное **Times New Roman**, 14 px, полужирным курсивом — точно то, что мы задали.

---

## Полный рабочий пример

Ниже приведена полностью готовая к запуску программа. Скопируйте её в `Program.cs` и нажмите **F5**.

```csharp
using System;
using System.IO;
using Aspose.Html;
using Aspose.Html.Drawing;

class Program
{
    static void Main()
    {
        // 1️⃣ Create a new empty HTML document
        HTMLDocument doc = new HTMLDocument();

        // 2️⃣ Define a font with both bold and italic styles
        Font font = new Font("Times New Roman", 14, WebFontStyle.Bold | WebFontStyle.Italic);

        // 3️⃣ Create a paragraph element and apply the font to its style
        HtmlElement paragraph = doc.CreateElement("p");
        paragraph.Style.Font = font;

        // 4️⃣ Set the paragraph's text content
        paragraph.InnerHtml = "Bold‑italic text rendered with WebFontStyle.";

        // 5️⃣ Add the paragraph to the document body
        doc.Body.AppendChild(paragraph);

        // 6️⃣ Save the resulting HTML
        string outputPath = Path.Combine(Environment.CurrentDirectory, "output.html");
        doc.Save(outputPath);
        Console.WriteLine($"HTML saved to: {outputPath}");
    }
}
```

### Ожидаемый результат

Открытие `output.html` должно показать:

> **Текст, оформленный полужирным курсивом с помощью WebFontStyle.** *(шрифт Times New Roman, 14 px, полужирный курсив)*

Если текст выглядит обычным, проверьте, установлен ли указанный шрифт в системе. В противном случае Aspose.HTML переключится на шрифт по умолчанию, но флаги стиля (полужирный & курсив) сохранятся.

## Часто задаваемые вопросы (FAQ)

**В: Можно ли использовать веб‑шрифт вместо системного?**  
О: Конечно. Замените `"Times New Roman"` на URL Google Font или любого другого хостинга и задайте `font.Source` соответствующим образом. Aspose.HTML автоматически добавит правило `@font-face`.

**В: Что делать, если нужны несколько абзацев с разными стилями?**  
О: Создайте отдельные объекты `Font` для каждого стиля, примените их к разным экземплярам `HtmlElement` и добавьте каждый в `body` или в контейнер.

**В: Работает ли это на .NET Core?**  
О: Да. Aspose.HTML кроссплатформенный, поэтому тот же код исполняется на Windows, Linux и macOS, если среда поддерживает .NET Standard 2.0+.

**В: Как добавить CSS‑классы вместо встроенных стилей?**  
О: Можно задать `paragraph.ClassName = "myClass"` и затем добавить блок `<style>` в `head` документа или подключить внешнюю таблицу стилей.

## Советы и подводные камни

- **Не жёстко прописывайте пути**: используйте `Path.Combine` и `Environment.CurrentDirectory` для переносимости кода.  
- **Освобождайте документ**: `HTMLDocument` реализует `IDisposable`. В продакшн‑коде оборачивайте его в `using`, чтобы своевременно освобождать ресурсы.  
- **Проверьте версию библиотеки**: в этом руководстве используется Aspose.HTML 23.12. В более старых версиях сигнатура конструктора `Font` может отличаться.  
- **Валидируйте HTML**: после сохранения прогоните файл через валидатор (например, W3C validator), чтобы убедиться в соответствии стандартам.
