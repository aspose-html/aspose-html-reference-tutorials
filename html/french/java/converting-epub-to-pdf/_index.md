## Questions fréquemment posées

**Q:** *Puis-je convertir des fichiers EPUB protégés en PDF ?*  
**A:** Oui. Aspose.HTML for Java peut ouvrir des EPUB chiffrés lorsque vous fournissez le mot de passe de déchiffrement approprié.

**Q:** *Et si j'ai besoin que le PDF soit conforme à PDF/A ?*  
**A:** Utilisez la classe `PdfSaveOptions` pour définir `Compliance` à `PdfCompliance.PdfA1b` (ou d'autres niveaux PDF/A) avant l'enregistrement.

**Q:** *Est-il possible d'extraire des images comme JPG ou PNG pendant la conversion ?*  
**A:** Absolument. En spécifiant une option d'enregistrement d'image ou en utilisant le fournisseur de flux personnalisé, vous pouvez diriger chaque page ou image vers le format JPG ou PNG.

**Q:** *Ai-je besoin d'une bibliothèque séparée pour l'extraction d'images ?*  
**A:** Non. La même API Aspose.HTML for Java gère à la fois la génération de PDF et l'extraction d'images, simplifiant le flux de travail.

**Q:** *Quelles versions de Java sont prises en charge ?*  
**A:** La bibliothèque prend en charge Java 8 et les versions ultérieures, y compris Java 11, Java 17 et les versions LTS suivantes.
