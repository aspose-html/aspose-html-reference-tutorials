# 在 Java 中執行 JavaScript – 完整指南：將元素附加至 Body

是否曾需要**在 Java 中執行 JavaScript**，卻不知如何在兩者之間搭橋？你並非唯一遇到此問題的人。許多 Java 開發者在想要進行**JavaScript DOM 操作**而不啟動完整瀏覽器時，常會卡住。好消息是？使用 Aspose.HTML，你可以執行腳本、**將元素附加至 body**，並儲存修改後的標記——全部只需純 Java 程式碼。

在本教學中，我們將逐步說明你需要了解的所有內容：載入 HTML 檔案、編寫一段**使用 JavaScript 修改 HTML**的簡短腳本、從 Java 執行該腳本，最後將變更持久化。完成後，你將擁有一個自包含、可直接執行的範例，隨時可放入任何專案中。

## 需要的環境

- **Java 8** 或更新版本（API 支援 Java 8+）
- **Aspose.HTML for Java** 函式庫（版本 22.12 或更新）
- 簡易的 HTML 範本檔案（我們會即時建立）
- IDE 或文字編輯器——不需要特殊工具，純文字編輯器亦可

> 小技巧：如果你使用 Maven，請將 Aspose.HTML 依賴加入 `pom.xml`。否則，只需將 JAR 檔案放入 classpath 即可。

## 步驟 1 – 載入 HTML 文件（為在 Java 中執行 JavaScript 做準備）

我們首先建立指向來源檔案的 `HTMLDocument` 實例。可將此物件視為 Java 與稍後腳本將操作之 DOM 樹之間的橋樑。

```java
import com.aspose.html.HTMLDocument;

// Step 1: Load the HTML document you want to manipulate
HTMLDocument htmlDoc = new HTMLDocument("YOUR_DIRECTORY/template.html");
```

> **為什麼這很重要：** `HTMLDocument` 類別會將檔案解析為 DOM 表示，供 Aspose 腳本引擎互動。若未先載入文件，腳本將無任何可修改的內容。

## 步驟 2 – 編寫 **將元素附加至 Body** 的 JavaScript

現在我們撰寫一段小程式碼，建立 `<p>` 元素並將其加入 `<body>`。這是典型的**JavaScript DOM 操作**，但我們將從 Java 執行它。

```java
// Step 2: JavaScript that adds a new paragraph to the body
String jsCode = ""
    + "var p = document.createElement('p');"
    + "p.textContent = 'Generated by JavaScript';"
    + "document.body.appendChild(p);";
```

> **為什麼這會有效：** `document.createElement` 會建立新節點，`textContent` 設定其內部文字，`appendChild` 將其附加到現有的 `<body>` 元素。此腳本純屬客戶端 JavaScript；Aspose.HTML 於無頭環境中執行它。

## 步驟 3 – 建立腳本引擎（能**從 Java 執行 JavaScript**的引擎）

Aspose.HTML 內建 `ScriptEngine`，可對已載入的 DOM 評估 JavaScript。它正是實際**從 Java 執行 JavaScript**的部件。

```java
import com.aspose.html.scripting.ScriptEngine;

// Step 3: Bind the engine to the document we just loaded
ScriptEngine scriptEngine = new ScriptEngine(htmlDoc);
```

> **為什麼需要它：** 引擎將腳本執行隔離，避免影響主 Java 程序。它亦會回傳 `ScriptExecutionResult`，讓你驗證是否成功。

## 步驟 4 – 執行腳本並檢查結果

所有設定完成後，我們終於執行腳本。`execute` 方法會回傳結果物件，告知腳本是否順利執行且無錯誤。

```java
import com.aspose.html.scripting.ScriptExecutionResult;

// Step 4: Run the JavaScript and verify it succeeded
ScriptExecutionResult execResult = scriptEngine.execute(jsCode);
if (execResult.isSuccess()) {
    System.out.println("Script executed successfully.");
} else {
    System.err.println("Script failed: " + execResult.getErrorMessage());
}
```

> **邊緣情況：** 若腳本拋出例外（例如語法錯誤），`isSuccess()` 會為 `false`，而 `getErrorMessage()` 會提供詳細資訊。務必在正式程式碼中檢查此結果。

## 步驟 5 – 儲存修改後的文件（完成**使用 JavaScript 修改 HTML**）

腳本改變 DOM 後，我們將變更寫回磁碟。最終檔案將包含新段落。

```java
// Step 5: Write the updated DOM back to an HTML file
htmlDoc.save("YOUR_DIRECTORY/modified.html");
System.out.println("Document saved.");
```

> **結果驗證：** 在任何瀏覽器開啟 `modified.html`。你應該會看到原始內容外加一行文字「Generated by JavaScript」。

## 步驟 6 – 完整範例（結合所有步驟）

以下是完整程式碼，你可以直接複製貼上至 `JsDomManipulation.java` 檔案。它包含所有匯入、錯誤處理與說明註解，便於閱讀。

```java
import com.aspose.html.HTMLDocument;
import com.aspose.html.scripting.ScriptEngine;
import com.aspose.html.scripting.ScriptExecutionResult;

public class JsDomManipulation {
    public static void main(String[] args) throws Exception {

        // 1️⃣ Load the HTML document to be manipulated
        HTMLDocument htmlDoc = new HTMLDocument("YOUR_DIRECTORY/template.html");

        // 2️⃣ Define JavaScript that adds a new paragraph to the <body>
        String jsCode = "var p = document.createElement('p');"
                      + "p.textContent = 'Generated by JavaScript';"
                      + "document.body.appendChild(p);";

        // 3️⃣ Create a script engine bound to the loaded document
        ScriptEngine scriptEngine = new ScriptEngine(htmlDoc);

        // 4️⃣ Execute the JavaScript and report success
        ScriptExecutionResult execResult = scriptEngine.execute(jsCode);
        if (execResult.isSuccess()) {
            System.out.println("Script executed successfully.");
        } else {
            System.err.println("Script failed: " + execResult.getErrorMessage());
            return; // Stop if the script didn't run
        }

        // 5️⃣ Save the modified document back to HTML
        htmlDoc.save("YOUR_DIRECTORY/modified.html");
        System.out.println("Document saved.");
    }
}
```

### 預期輸出

執行程式會輸出：

```
Script executed successfully.
Document saved.
```

而 `modified.html` 現在的內容類似於：

```html
<!DOCTYPE html>
<html>
<head>
    <title>Template</title>
</head>
<body>
    <h1>Original Content</h1>
    <p>Generated by JavaScript</p>
</body>
</html>
```

## 常見問題與注意事項

| Question | Answer |
|----------|--------|
| **我可以依序執行多個腳本嗎？** | 可以。只要再次呼叫 `scriptEngine.execute()` 並傳入新的腳本字串即可。同一個 `HTMLDocument` 實例會保留其狀態。 |
| **如果 HTML 檔案缺少 `<body>` 標籤會怎樣？** | Aspose.HTML 會在解析時自動建立 `<body>` 元素，因此腳本仍能執行。但為了得到可預測的結果，建議使用結構完整的範本。 |
| **此方式與 Java 17 相容嗎？** | 絕對相容。此函式庫支援 Java 8 以上，較新版本的執行環境亦可使用。 |
| **我需要完整的瀏覽器環境嗎？** | 不需要。Aspose.HTML 提供輕量且無頭的引擎，能解讀標準 DOM API，無需啟動 Chrome 或 Firefox。 |
| **如何處理大型 HTML 檔案？** | 將文件以串流方式載入（`new HTMLDocument(InputStream)`），以避免一次將整個檔案載入記憶體。 |
