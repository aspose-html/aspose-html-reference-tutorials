## 常見問題與解決方案
- **PDF 中缺少字型** – 確認伺服器已安裝所需字型，或使用 `PdfSaveOptions` 內嵌字型。  
- **大型 EPUB 檔案導致記憶體壓力** – 採用串流處理（`MemoryStream` → `FileStream`）以降低堆積使用量。  
- **Canvas 渲染為空白** – 確認在呼叫轉換 API 前，canvas 已完整繪製。
