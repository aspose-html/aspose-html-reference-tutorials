## 如何使用 Aspose.HTML for Java 轉換 html 為 pdf？
Aspose.HTML for Java 處理現代 HTML、CSS3 與 JavaScript 的渲染工作，然後直接將輸出串流至 PDF 檔案。此過程通常包含三個步驟：

1. **載入 HTML 來源** – 從檔案、URL 或字串。  
2. **設定轉換選項** – 如頁面大小、邊距或字型嵌入。  
3. **將結果儲存為 PDF** – 使用 `PdfSaveOptions` 類別。

這些步驟讓您在保持程式碼簡潔易維護的同時，擁有精細的控制。

## 常見問題與解決方案
- **PDF 中缺少字型** – 確保伺服器已安裝所需字型，或使用 `PdfSaveOptions` 進行嵌入。  
- **大型 EPUB 檔案導致記憶體壓力** – 使用串流方式處理（`MemoryStream` → `FileStream`）以減少堆積使用量。  
- **Canvas 渲染呈現空白** – 在呼叫轉換 API 前，確認 canvas 已完整繪製。
