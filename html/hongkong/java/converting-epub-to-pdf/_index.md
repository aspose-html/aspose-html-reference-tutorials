**A:** 可以。Aspose.HTML for Java 在您提供相應的解密密碼時，能開啟加密的 EPUB。  

**Q:** *如果我需要 PDF 符合 PDF/A 標準怎麼辦？*  
**A:** 在儲存前使用 `PdfSaveOptions` 類別將 `Compliance` 設為 `PdfCompliance.PdfA1b`（或其他 PDF/A 級別）。  

**Q:** *在轉換過程中能否提取 JPG 或 PNG 等圖像？*  
**A:** 完全可以。透過指定圖像儲存選項或使用自訂串流提供者，您可以將每頁或圖像導出為 JPG 或 PNG 格式。  

**Q:** *我需要額外的函式庫來提取圖像嗎？*  
**A:** 不需要。相同的 Aspose.HTML for Java API 同時處理 PDF 產生與圖像提取，簡化工作流程。  

**Q:** *支援哪些 Java 版本？*  
**A:** 此函式庫支援 Java 8 及以上版本，包括 Java 11、Java 17 以及後續的 LTS 版本。
