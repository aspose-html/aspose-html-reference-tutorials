## So führen Sie die svg to pdf java‑Konvertierung mit Aspose.HTML für Java durch
Um eine SVG‑Datei mit Aspose.HTML für Java in PDF zu konvertieren, laden Sie das SVG in ein `SVGDocument`, konfigurieren `PdfSaveOptions` zur Steuerung von Qualität und Layout und rufen anschließend die `save`‑Methode auf, um das PDF zu schreiben. Dieser unkomplizierte Arbeitsablauf kann in Schleifen für die Batch‑Verarbeitung eingebettet und in jede Java‑Anwendung integriert werden.

Laden Sie das SVG, konfigurieren Sie die PDF‑Optionen und speichern Sie das Ergebnis – alles in vier knappen Schritten.

### Direkte Antwort
Laden Sie Ihr SVG mit `new SVGDocument("input.svg")`, konfigurieren Sie `PdfSaveOptions` (z. B. setzen Sie `jpegQuality` auf 90) und rufen Sie dann `document.save("output.pdf", saveOptions)` auf. Dieser einzeilige Arbeitsablauf konvertiert die Vektorgrafik in ein PDF, wobei Layout, Farben und Schriftarten erhalten bleiben.

### Pakete importieren
Die Klasse `SVGDocument` ist Aspose.HTMLs Darstellung einer SVG‑Datei im Speicher. Importieren Sie die erforderlichen Namespaces, bevor Sie beginnen:

**Definitionsanker:** `SVGDocument` ist die Kernklasse, die SVG‑Inhalt parst und für die weitere Verarbeitung hält.

```
import com.aspose.html.SVGDocument;
import com.aspose.html.rendering.pdf.PdfSaveOptions;
import com.aspose.html.rendering.pdf.PdfDevice;
```

### Schritt 1: SVG‑Dokument laden
`SVGDocument` liest die XML‑basierte SVG‑Datei und bereitet sie für das Rendering vor.

**Definitionsanker:** `SVGDocument` lädt die SVG‑Quelle und stellt eine DOM‑ähnliche API zur Manipulation bereit.

```
SVGDocument svgDoc = new SVGDocument("path/to/input.svg");
```

### Schritt 2: PDF‑Speicheroptionen konfigurieren
`PdfSaveOptions` ermöglicht die Steuerung von Ausgabequalität, Seitengröße und Kompression.

**Definitionsanker:** `PdfSaveOptions` fasst alle PDF‑spezifischen Einstellungen wie Bildqualität und Seitendimensionen zusammen.

```
PdfSaveOptions saveOptions = new PdfSaveOptions();
saveOptions.setJpegQuality(90);               // Set JPEG quality to 90%
saveOptions.setPageSize(PdfDevice.PageSize.A4); // Use A4 page size
```

### Schritt 3: Ausgabepfad festlegen
```
String outputPath = "path/to/output.pdf";
```

### Schritt 4: SVG in PDF konvertieren
Die `save`‑Methode führt die eigentliche Konvertierung durch.

**Definitionsanker:** `document.save` schreibt den gerenderten Inhalt im angegebenen Format unter Verwendung der bereitgestellten Optionen.

```
svgDoc.save(outputPath, saveOptions);
```

Herzlichen Glückwunsch! Sie haben erfolgreich **ein PDF aus SVG erstellt** mit Aspose.HTML für Java. Das PDF befindet sich an dem von Ihnen in `outputPath` angegebenen Pfad.
