## Häufige Probleme und Lösungen
- **Missing fonts in PDF** – Stellen Sie sicher, dass die erforderlichen Schriftarten auf dem Server installiert sind oder betten Sie sie mit `PdfSaveOptions` ein.  
- **Large EPUB files cause memory pressure** – Verwenden Sie stream‑basierte Verarbeitung (`MemoryStream` → `FileStream`), um den Heap‑Verbrauch zu reduzieren.  
- **Canvas rendering appears blank** – Vergewissern Sie sich, dass das Canvas vollständig gezeichnet wurde, bevor Sie die Konvertierungs‑API aufrufen.

A: Ja, die Rendering‑Engine unterstützt die meisten CSS3‑Eigenschaften, einschließlich Flexbox und Grid‑Layouts.

**Q: Wie automatisiere ich das Ausfüllen mehrerer HTML‑Formulare?**  
A: Verwenden Sie die `Form`‑API, um ein Dokument zu laden, Feldwerte programmgesteuert zu setzen und das Ergebnis zu speichern.

**Q: Ist es möglich, eine HTML‑Seite direkt zu SVG zu konvertieren?**  
A: Absolut – die `HtmlToSvgConverter`‑Klasse übernimmt diese Konvertierung mit hoher Treue.

**Q: Was ist der beste Weg, ein großes HTML‑Canvas zu PDF zu konvertieren?**  
A: Rendern Sie das Canvas zuerst zu einem Bitmap, verwenden Sie dann `PdfSaveOptions`, um das Bild einzubetten, oder nutzen Sie die integrierte Canvas‑zu‑PDF‑Methode für Vektor‑Ausgabe.
