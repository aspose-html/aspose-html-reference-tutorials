## Häufig gestellte Fragen

**Q: Kann ich eine einzelne Datei aus einem großen ZIP entfernen, ohne das gesamte Archiv zu extrahieren?**  
A: Ja, der ZIP Archive Message Handler ermöglicht es Ihnen, gezielt bestimmte Einträge direkt zu löschen.

**Q: Wie füge ich neue Dateien zu einem bestehenden ZIP mit Aspose.HTML hinzu?**  
A: Öffnen Sie das Archiv mit dem Handler, verwenden Sie die `add`‑Methode, um die neue Datei einzufügen, und speichern Sie die Änderungen.

**Q: Ist es möglich, ein ZIP‑Archiv zu lesen, ohne es zuerst zu extrahieren?**  
A: Absolut. Der Handler bietet Streaming‑APIs, mit denen Sie Dateien bei Bedarf lesen oder bereitstellen können.

**Q: Muss ich den ZIP‑Passwortschutz selbst handhaben?**  
A: Aspose.HTML unterstützt verschlüsselte ZIPs; Sie können das Passwort beim Öffnen des Archivs angeben.

**Q: Welche Auswirkungen hat das Entfernen von Dateien auf die Performance?**  
A: Der Vorgang wird im Speicher durchgeführt und schreibt nur die modifizierten Teile zurück, wodurch I/O minimiert wird.
