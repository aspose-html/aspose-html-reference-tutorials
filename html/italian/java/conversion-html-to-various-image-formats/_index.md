## Come **convertire html gif**, **convertire html jpeg** e **convertire html png** con Aspose.HTML
Di seguito troverai istruzioni concise passo‑passo per ciascun formato di destinazione. Gli snippet di codice sono identici tranne per l'estensione del file di output; è sufficiente modificare il formato di `ImageSaveOptions`.

### Passo 1: Configurare il renderer HTML
Crea un'istanza di `HtmlRenderer` e carica la sorgente HTML (file, URL o stringa).

### Passo 2: Scegliere il formato immagine desiderato
Seleziona una delle sottoclassi di `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions`, ecc.

### Passo 3: Renderizzare e salvare
Chiama `renderer.renderToFile(outputPath, saveOptions)` per generare l'immagine.

*Gli stessi tre passaggi si applicano a BMP, TIFF e a qualsiasi altro formato supportato.*
