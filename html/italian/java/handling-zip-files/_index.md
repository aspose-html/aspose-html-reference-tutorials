## Come rimuovere file da zip usando Aspose.HTML per Java

Il `ZIP Archive Message Handler` è una classe che fornisce la manipolazione in‑memoria degli archivi ZIP. Carica lo ZIP con il **ZIP Archive Message Handler**, individua l'elemento che desideri eliminare, invoca il metodo `remove` e poi salva l'archivio. Questo rimuove il file senza estrarre l'intero ZIP, riducendo i tempi di I/O e mantenendo l'applicazione reattiva.

Aspose.HTML fornisce un **ZIP Archive Message Handler** che agisce come un assistente personale per i tuoi pacchetti compressi. Con poche chiamate di metodo puoi aprire uno ZIP, individuare l'elemento da eliminare e rimuoverlo—tutto senza estrarre manualmente l'archivio prima. Questo approccio riduce il sovraccarico di I/O e mantiene l'applicazione reattiva.

## Come aggiungere file a zip java con Aspose.HTML

Apri l'archivio con l'handler, chiama `add` (o `replace`) per inserire una nuova voce e salva le modifiche. L'handler aggiorna lo ZIP in‑memoria, così non è mai necessario ricreare l'archivio da zero.
