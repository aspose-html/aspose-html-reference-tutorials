## Problemi Comuni e Soluzioni
- **Font mancanti nel PDF** – Assicurati che i font richiesti siano installati sul server o incorporali usando `PdfSaveOptions`.  
- **File EPUB di grandi dimensioni causano pressione sulla memoria** – Usa l'elaborazione basata su stream (`MemoryStream` → `FileStream`) per ridurre l'uso dell'heap.  
- **Il rendering del canvas appare vuoto** – Verifica che il canvas sia stato disegnato completamente prima di chiamare l'API di conversione.
