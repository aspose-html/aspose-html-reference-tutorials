## Hogyan **convert html gif**, **convert html jpeg**, és **convert html png** Aspose.HTML‑el

Az alábbiakban tömör, lépésről‑lépésre útmutatót talál minden célformátumhoz. A kódrészletek azonosak, csak a `ImageSaveOptions` formátumát kell módosítani.

### 1. lépés: Állítsa be a HTML renderert
Hozzon létre egy `HtmlRenderer` példányt, és töltse be a HTML forrást (fájl, URL vagy karakterlánc).

### 2. lépés: Válassza ki a kívánt képfájltípust
Válasszon egy `ImageSaveOptions` alosztályt – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions` stb.

### 3. lépés: Renderelés és mentés
Hívja meg a `renderer.renderToFile(outputPath, saveOptions)` metódust a kép előállításához.
