## Como **convert html gif**, **convert html jpeg** e **convert html png** com Aspose.HTML
Abaixo você encontrará instruções concisas passo a passo para cada formato de destino. Os trechos de código são idênticos, exceto pela extensão do arquivo de saída; basta alterar o formato em `ImageSaveOptions`.

### Etapa 1: Configurar o renderizador HTML
Crie uma instância de `HtmlRenderer` e carregue a fonte HTML (arquivo, URL ou string).

### Etapa 2: Escolher o formato de imagem desejado
Selecione uma das subclasses de `ImageSaveOptions` – `GifSaveOptions`, `JpegSaveOptions`, `PngSaveOptions`, etc.

### Etapa 3: Renderizar e salvar
Chame `renderer.renderToFile(outputPath, saveOptions)` para gerar a imagem.

*Os mesmos três passos se aplicam a BMP, TIFF e qualquer outro formato suportado.*
