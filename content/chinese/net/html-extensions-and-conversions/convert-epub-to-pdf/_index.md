
- 使用`Converter.ConvertEPUB`使用自定义选项将 EPUB 转换为 PDF 的方法。

```csharp
                System.IO.MemoryStream result = new System.IO.MemoryStream();

            }
            public void ReleaseStream(System.IO.Stream stream)

                //在这里您可以释放充满数据的流，例如将其刷新到硬盘驱动器

                //释放资源
```
