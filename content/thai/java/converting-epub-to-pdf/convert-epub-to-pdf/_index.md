## ตั้งค่าตัวเลือกการแปลง PDF

 สร้างอินสแตนซ์ของ`PdfSaveOptions` เพื่อระบุตัวเลือกการแปลงสำหรับเอาต์พุต PDF

```java

```

ตัวเลือกเหล่านี้ช่วยให้คุณสามารถกำหนดการตั้งค่าต่างๆ สำหรับเอาต์พุต PDF เช่น ขนาดหน้าและระยะขอบ
